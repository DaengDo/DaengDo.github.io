>변수가 왜 필요할까?

- 값(value)에 비해 변수(variable)는 보다 일관적이고 확장된 개념의 기능을 생각할 수 있다.

<br>

>컴퓨터에서는 어떻게 기능할까?

- 컴퓨터의 cpu는 계산을 해주고 메모리는 데이터를 저장한다.<br> `x + y = z` 의 예시를 들어보자
  - x, y z 는 메모리가 +, = 의 연산 과정은 cpu가 작동한다.

<br>

>변수의 도입으로 프로그래밍 작업이 편리해진다.

- 임의접근메모리(RAM)는 주소를 갖고있고 cpu는 그 주소를 통해 빠르게 데이터에 접근 할 수 있다.

- 변수 개념의 도입으로 이 주소를 거처야하는 과정을 생략할 수 있다.

```c
#include <stdio.h>

int main(void)
{
    int x;
    int y;
    int z; 
          // 변수를 선언하고
    x = 1;
    y = 2;
          // 변수에 값을 대입하고
    z = x + y;
          // 연산을하고
    printf("Result is %i", z);
          //연산의 결과를 변수에 대입하는 그 어느 과정에서도 프로그래머가 메모리의 주소를 직접 알고있을 필요가 없다.
    return 0;
}
```
<BR>

`인간 프로그래머에게 익숙한 문자로 마치 그 메모리에 직접 접근할 수 있는것처럼 편리함을 제공하는 것이다.`
