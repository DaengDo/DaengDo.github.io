# 정수와 실수

## 정수 ( Integers )

- 컴퓨터 내부적으로 signed unsigned integer로 구분한다.<br>Unsigned : 0과 양의 정수(부호가 없다, unsigned)<br>Signed : 음의 정수, 0, 양의 정수 모두 포함
- (사람은 10진수를 사용할지몰라도) 내부적으로 2진수로 표현된다.

<br>

## 실수 ( Real numbers )

- 정수 포함, 정수 사이의 값들을 포함 시킬 수 있다.<br>
- 자료형을 미리 정해두는 것이기 때문에 숫자 2와 2.0은 의미상 같지만 내부적으로 구분되어있다.
- 내부적으로 부 _(한자:浮뜰부)_ 동 소수점 (floating point) 표현법을 사용한다.<br>
  0.314<span style="color:red">E</span>1 = 0.314 X <span style="color:red">10</span>^1<br>
  3.14<span style="color:red">E</span>123 = 3.14 X <span style="color:red">10</span>^123<br>
  0.314<span style="color:red">E</span> - 1 = 0.314 X <span style="color:red">10</span>^-1<br>
  3.14 = 3.14E0 = 3.14e0 = 0.314E1 = 31.4E-1<br>
  E는 지수(Exponent) 의 줄임 표현법이다.
- 내부적으로 integer와 마찬가지로 2진수로 표현된다.

<br>

- 실수 자료형이 필요할때 float 과 double 메모리 크기를 꼭! 고려해야한다.

<br><br><br>

## Floating-Point Types

| Type          | (min)Storage size | Value range         | Precision         |
| ------------- | ----------------- | ------------------- | ----------------- |
| float         | 4 byte            | 1.2E-38 ~ 3.4E+38   | 6 decimal places  |
| (long) double | 8 byte            | 2.3E-308 ~ 1.7E+308 | 15 decimal places |

long double 의 경우에는 32bit 64bit 운영체제에 따라 크기가 다른듯하다 sizeof 연산자를 활용해보자
