# 불리언형 (boolean types)

- 고전적인 c 에서는 불리언이 존재하지 않았다.<br>현재에는 \_Bool 이라는 키워드가 존재한다.
- True 혹은 False 의 값만 갖는다.
- 내부적으로는 정수형이기 때문에 0이면 false 1이면 true 값을 갖는다.<br>1 bit 면 표현 할 수 있지만, 자료형이 가질 수 있는 가장 작은 값이 1byte 이기 때문에 1byte를 이용한다.
  - byte 가 메모리에서 주소를 배정받을 수 있는 최소 단위이기 때문이다.
  - 나머지 7 bit도 활용하는 방법이 있다. (뒤의 bit연산 수업에서 알려주겠다고 함 아 고구마 먹은거같아 답답해)
- 출력을 할때도 별다른 형식 지정자가 없다. 정수형의 형식 지정자를 활용한다. (%d)
  - 표준에 stdbool.h 가 추가된 이후 전처리기에서 포함을 시키면 예약어 true와 false 를 적을 수 있게 된다. <br>(그러나 결국 내부적으로 false는 0 true는 1로 처리함, 근본은 같다.)
