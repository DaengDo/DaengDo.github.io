# 8진수와 16진수

- 8/16진수를 쓰면 2진수의 자릿수를 줄일 수 있어서 많이 활용한다.

| 표기 | 2진수 | 8진수   | 16진수      |
| ---- | ----- | ------- | ----------- |
| 입력 | 0b ~  | 0 ~     | 0x ~        |
| 출력 |       | %o, %#o | %X, %x, %#x |

<br>

예시

- 3(10진수) = 0b11(2진수) = 03(8진수) = 0x3 (16진수)
