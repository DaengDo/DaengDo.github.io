# 사실과 거짓

- c언어에서는 0 이면 false고 나머지는 true 다.
- `#define TRUE 1` `#define FALSE 0` 으로 전처리기에서 선언하기도 했었으나, 불리언이라는 자료형이 추가되었기 때문에 이제는 안씀
```c
#include <stdio.h>

int main()
{
  while(1){
    // 오픈소스같은데 보면 종종있는 코드 형식
    // 주로 서버 처럼 계속 돌아가는 것이 전제가 되는 코드에서
    // 무한루프를 돌리는 식의 트릭을 사용한다.
    // 나올때는 반복문 안에서 깨고 나옴
  }
}
```
