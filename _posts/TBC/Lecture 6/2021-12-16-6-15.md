{: .small}

# Task 1

아래 문구를 출력하시오

```
A B C D E F G H I J
A B C D E F G H I J
A B C D E F G H I J
A B C D E F G H I J
A B C D E F G H I J
```

## Solution

```c
#include <stdio.h>

#define LAST_CHAR 'K'
#define NUM_ROWS 5

int main()
{
  char c;
  int i;

  for(i = 0; i < NUM_ROWS; i++)
  {
    for(c = 65; c < LAST_CHAR; c++)
    {
      printf("%c ", c);
    };
    printf("\n");
  }

  return 0;
}
```

c 스타일답게 상수를 `const`대신 `#define`을 사용해보았다.

# Task 2

아래 문구를 출력하시오

```
A 
A B
A B C
A B C D
A B C D E
A B C D E F
A B C D E F G
A B C D E F G H
A B C D E F G H I
A B C D E F G H I J
```

## Solution

```c
#include <stdio.h>

int main()
{
  const char FIRST_CHAR = 'A';
  const int ROWS = 10;
  int i;  
  char c = 65;

  for (i = 0; i < ROWS; i++)
  {
    for(c = FIRST_CHAR; c <= FIRST_CHAR + i; c++)
    {
      printf("%c ", c);
    }
    printf("\n");
  }

  return 0;
}
```
첫번째 루프가 두번째 루프의 조건에 관여를 한 경우이다.

# Task 3

아래 문구를 출력하시오

```
A B C D E F G H I J
B C D E F G H I J
C D E F G H I J
D E F G H I J
E F G H I J
F G H I J
G H I J
H I J
I J
J
```

## Solution

```c
#include <stdio.h>

int main()
{
  const char LAST_CHAR;
  printf("Please enter \"Captial letter(Only A to Z)\" you want : ");
  scanf("%c", &LAST_CHAR);
  const char FIRST_CHAR = 'A';
  const int ROWS = LAST_CHAR - FIRST_CHAR + 1;
  int i;
  char j;

  for (i = 0; i < ROWS; i++)
  {
    for(j = FIRST_CHAR + i; j <= LAST_CHAR; j++)
    {
      printf("%c ", j);
    }
    printf("\n");
  }

  return 0;
}
```
첫번째 루프가 두번째 루프의 초기화에 관여를 한 경우이다.<BR> 추가로 사용자가 마지막 글자를 정할 수 있게끔 커스터마이징을 했다.
