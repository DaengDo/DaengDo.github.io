# while 반복 루프에서 scanf()의 반환값 사용하기

- scanf()의 반환값(return)은 입력받은 변수의 갯수이다.
- 반복되는 코드는 유지보수 측면에서도 비효율적이고 실수하기 쉽다.<br>간결하게 코드를 작성해 최대한 반복되는 코드를 줄여야 한다.

```c
#include <stdio.h>

int main()
{
    int num, sum = 0;

    printf("Enter an integaer (q to quit) : ");

    while(scanf("%d", &num) == 1)
    {
        sum = sum + num;

        printf("Enter an integaer (q to quit) : ");
    }

    printf("Sum = %d\n", sum);

    return 0;
}
```
