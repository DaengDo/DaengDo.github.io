```c
#include <stdio.h>

void say_hello(void); // prototyping, declaration (선언)

int main()
{
    say_hello();
    say_hello();
    say_hello();
    say_hello();
    say_hello();
    
    return 0;
}
void say_hello(void) // function definition (정의)
{
    printf("Hello, World!\n");

    // return; 리턴 타입이 void 인 경우에는 return; 을 생략할 수 있다.
}
```

C는 옛날 언어라서 main 함수 아래에 선언된 함수를 이용하려면 먼저 선언되거나 정의 되어 있어야 한다.
