# 예약어(keyword)

- 컴퓨터 프로그래밍 언어에서 이미 문법적인 용도로 사용되고 있기 때문에 식별자로 사용할 수 없는 단어이다.
- 메인 함수 아래에 적은 함수는 메인 함수가 인지하지 못하므로 메인 함수 위에 함수 선언을 하거나 전방 선언 (forward declaration)을 해야한다. (가독성면에서는 forward declaration이 나은듯하다.)
  - https://stackoverflow.com/questions/2632601/why-are-forward-declarations-necessary
  - 하지만 전방 선언조차도 부득이할 경우 임시방편으로 활용하는 방안이며 되도록 지양한다고 한다.

```c
#include <stdio.h>
#include <stdbool.h>

int main()
{
    int printf = 1;

    // int bool = 2;

    return 0;
}
```

변수 이름을 짓거나 함수 이름을 지을때 예약어나 키워드를 쓰면 안된다!!!
