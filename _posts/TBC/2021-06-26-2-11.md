```c
#include <stdio.h>

int main()
{
    int n1, n2, n3, n4;

    //  n5 = 3; syntax error

    n1 = 1;
    n2 = n1 * n1;
    n3 = n2 * n1;
    n4 = n2 * n1; // semantic error

    printf("%d",n2);

    return 0;
}
```

# 문맥 오류(semantic errors) 문법 오류(syntax errors)

- 문법 오류는 컴파일러가 찾아주기 때문에 인지하기 쉽다.
- 문맥 오류는 컴파일러가 찾아주지 않기 때문에 찾기 힘들다.
  - 문맥 오류가 있을 경우 알려줄 수 있는 예제인지 확인해야한다.

문법 오류는 컴퓨터가 알려주지만 문맥 오류는 컴퓨터가 알려주지않는다!
