```c
int main(void)  /* main 함수의 입력은 ()에 들어가고,
                  출력의 데이터 타입이
                  int(정수형)이라는 뜻이다.  
                  프로그램을 시작할때는 main 함수를
                  시작점으로 삼는다.
                  main 의 이름은 바꿀 수 없고,
                  단 하나만 존재해야한다. */

{               /* 메인 함수의 영역을 중괄호로 표시,    
                   실제로 수행하고싶은 부분은 
                   이곳에 적으면 된다. */
    int x, y, z;
    
    x = 1;
    y = 2;

    z = x + y;
    
    
    return 0;   /* 운영체제가 main 이라는 함수를
                   실행시킨 후 나올 출력값을 
                   int형(정수형)인 0으로 준다는 뜻이다. 운영체제에게 0을 return 시키는것은
                   모든 일이 끝났다는것을 뜻하기도한다. */

                /* return 아래에 적는것은
                   아무것도 실행이 되지 않는다.
                   (이미 운영체제에게
                   메인 함수가 끝났다고 알려줬기 때문에) */
}
```

# printf() 함수의 기본적인 사용법

- printf는 print formatted 의 줄임말이다.
- 직접 선언하거나, 헤더파일에서 갖고오면 된다.
  - printf 함수는 stdio.h 에서 갖고온 함수이다.

# escape sequence

- "\\" 표기는 컴파일러에게 명령점이 된다.<BR>(컴파일러가 착착착 읽다가 \\ 보이는 순간 명령어를 인식함!)
