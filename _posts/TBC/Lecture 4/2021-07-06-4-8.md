# 변환 지정자의 수식어들

## %[flags][width][.precision][length]specifier

- ex) printf("%+10.5hi", 256);

> flags

| flags      | description                                                                      |
| ---------- | -------------------------------------------------------------------------------- |
| `-`        | 왼쪽 정렬, 붙이지 않으면 오른쪽 정렬                                             |
| `+`        | 양수나 음수 출력, 붙이지 않으면 음수일 경우에만 - 출력                           |
| `#`        | o, x 혹은 X 표기할때 0,0x,0X 를 출력,<br>a,A,e,E,f,F,g,G 표기할때도 마찬가지     |
| `0`        | 빈칸있을경우 대신에 0을 채워넣음                                                 |
| width      | description                                                                      |
| (숫자)     | 숫자를 넣으면 초과하는 자릿수만큼 앞에 빈칸이 생김                               |
| `*`        | 예제로 설명                                                                      |
| .precision | description                                                                      |
| .숫자      | 숫자를 넣으면 그 숫자만큼 최소 출력되는 정밀도를 보정 숫자가 모자라면 0으로 채움 |
| `.*`       | 예제로 설명                                                                      |
