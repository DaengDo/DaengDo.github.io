# strlen() 함수

- 문자열의 길이를 잴 때 쓰는 함수이다.(string length)
- null character(\0)를 제외하고 문자의 숫자를 재준다.
- 동적할당시 strlen 함수는 역시 문자열의 길이(null character 제외)를 출력해주고, sizeof 함수 사용시 동적할당된 메모리 자료형의 크기를 출력해준다.
