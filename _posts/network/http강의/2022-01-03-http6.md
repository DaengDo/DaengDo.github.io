# 1. 1xx(Informational)
요청이 수신되어 처리중

<BR>

# 2. 2xx(Successful)
요청 정상 처리

<BR>

# 3. 3xx(Redirection)
요청을 완료하려면 추가 행동이 필요
<BR>

# 4. 4xx(Client Error)
클라이언트 오류, 잘못된 문법 등으로 서버가 요청을 수행할 수 없음
<BR>

# 5. 5xx(Server Error)
서버 오류, 서버가 정상 요청을 처리하지 못함
