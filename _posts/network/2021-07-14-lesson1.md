# 1. 컴퓨터 네트워크란?

- 컴퓨터 간의 연결을 컴퓨터 네트워크라고 부른다.<br>컴퓨터 간의 데이터 전송, 웹 사이트 열람, 메일 송/수신과 같은 일을 할 수 있다.
- 인터넷은 전 세계의 큰 네트워크부터 작은 네트워크까지를 연결하는 거대한 네트워크다.

# 2. 패킷(packet)이란?

_네트워크나 인터넷에서 데이터를 주고받으려면 규칙이 있어야 한다._

- 네트워크를 통해 전송되는 데이터의 작은 조각이다.
  - 큰 데이터를 그대로 보내면 그 데이터가 네트워크의 `대역폭`을 너무 많이 점유해서 다른 데이터의 흐름을 막을 위험이 있다.
    > 대역폭 (bandwidth) :<br>일반적으로는 네트워크에서 이용 가능한 최대 전송 속도로 정보를 전송할 수 있는 단위 시간당 전송량을 말한다.
  - 송신 측에서는 데이터를 패킷으로 나누어 각 패킷에 순서대로 번호를 붙여서 보내며, 수신 측에서 번호에 맞게 정렬한다.
