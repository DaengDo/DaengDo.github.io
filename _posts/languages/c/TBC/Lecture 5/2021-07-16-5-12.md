# 함수의 인수와 매개변수 (Arguments VS. Parameters)

| 단어      | 번역      | 의미                                 |
| --------- | --------- | ------------------------------------ |
| Parameter | 매개변수  | 함수와 메서드 입력 변수 (variable)명 |
| Argument  | 인자,인수 | 함수와 메서드의 입력 값 (value)      |

<br><br>

- Parameter 는 선언 부분에 있는 것이고, <br>Argument 는 실제 사용시의 데이터를 의미한다.
- Argument 와 Parameter가 수식어까지 붙어서 혼용을 하자 C99 에서 정리함
