# 나머지 연산자 (modulus operator)

> 나머지 연산자?

- %로 표기한다.
  - int a = 13 % 5; 에서 a 의 값은 3
  - int b = 90 % 60; 에서 b 의 값은 30

> 음수 값을 나누기,나머지 연산자로 활용할때 주의해야한다.
