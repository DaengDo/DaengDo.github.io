# 표현식과 문장 (expressions and statements)

> 표현식(expressions)

- 표현식은 값을 갖는다.

> 문장 (Statements)

- 문장의 끝은 세미 콜론으로 표기해야한다.

> side effects and sequence points

**statements**

- int x, y, apples; // declaration statement <br>
- apples = 3; // assignment statement <br>
- ; // null statement <br>
- 1+2;<br>
  x=4;<br>
  ++x;<br>
  x=1+(y=5); // y = 5 is subexpression<br>
- while (x++ < 10) // while statement (strucbtured statements)
- printf("%d\n", y); // function statement<br>
- return 0; // return statement<br>

<br>

**Side Effects and Sequence Points**

- x = 4; // main intent is evaluating expressions

- while (x++ < 10) // (x++ < 10) is a full expression.

- y = (4 + x++) + (6 + x++); <br> // Not (4 + x++) nor (6 + x++) is a full expression.
