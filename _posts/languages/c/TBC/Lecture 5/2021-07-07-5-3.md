# 더하기, 빼기, 부호 연산자들

- 이항 연산자(binary operator)
  - ex) 3 - 2 // 피연산자 2개
- 단항 연산자(unary operator)
  - ex) -16 // 피연산자 1개
- 복합
  - ex) -(12-11)
