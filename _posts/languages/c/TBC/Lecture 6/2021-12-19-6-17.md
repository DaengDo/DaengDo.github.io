숫자 5개를 입력받고 그 숫자들을 합해주는 프로그램 작성하기

조건: for문, 배열 사용할 것

```c
#include <stdio.h>

#define ARR_LENGTH 5

int main()
{
  int arr[ARR_LENGTH];
  int arr_sum = 0;
  int arr_value;

  printf("Enter %d integers : ", ARR_LENGTH);

  for(int i = 0; i<ARR_LENGTH; ++i)
  {
    scanf("%d", &arr[i]);
  }

  for(int j = 0; j<ARR_LENGTH; ++j)
  {
    arr_sum += arr[j];
  }

  printf("Sum = %d", arr_sum);

  return 0;
}
```

재미있는 점<BR>
- 최근 컴퓨팅 기법에서는 멀티쓰레딩이나 gpu가속이 많이 발달되고 예전과 달리 데이터양이 많기 때문에 for 루프를 여러개로 쪼개는게 효율적인 경우가 많다고 한다.
