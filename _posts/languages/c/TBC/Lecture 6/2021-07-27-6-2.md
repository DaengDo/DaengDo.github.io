# pseudo code (의사 코드)

(pseudo : 허위의, 가짜의; 모조의)

- 특정 언어의 문법에 대해 생각하지 않고 논리 자체에 집중하기 위해 인간의 언어로 적는 것이다.

```c
#include <stdio.h>
int main()
{
    int num, sum = 0;
    int status;

    printf("Enter an integer : ");
    status = scanf("%d", &num);

    while(status == 1)
    {
        sum = sum + num;

        printf("Enter an integer : ");
        status = scanf("%d, &num);
    }

    printf("Sum = %d", sum);

    return 0;
}
```

위의 예시를 pseudo code 로 적으면 아래와 같다.

<div class="notice--primary" markdown="1">
sum을 0으로 초기화한다.             initialize sum to 0
사용자에게 안내한다.                Prompt user
사용자에게 입력을 받는다.           Read input
그 입력이 정수인 동안에는           While the input is an integer
  그 입력을 sum에 더한다.             Add the input to sum,
  사용자에게 안내한다.                Prompt user,
  사용자에게 다음번 입력을 받는다.    Then read next input
sum을 출력한다.                     After input complete, print sum
</div>

pseudo code 는 정해진 문법이 있다기보다는 대강 아 이정도면 이해할 수 있어! 정도면 충분하다.
논리성을 잃지않는 범위내에서 편하게 작성하면 된다.

pseudo code의 두번째 중요성

- 언어 중립적으로 알고리즘, 논리 흐름을 표현할 수 있다.
- 어떤 언어를 사용하던 pseudo code를 보면 서로 의사소통이 가능하다.

예를 들어 pseudo code를 가장 많이 볼때는 알고리즘을 공부할때이다.

pseudo code를 한글로 적는 경우는 거의 없으며, 주로 영어로 쓴다.

tip)

```c
while (scanf("%d", &num) == 1)
{
  // do something with num
```
