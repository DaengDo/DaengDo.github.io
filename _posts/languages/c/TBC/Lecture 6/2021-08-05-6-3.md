# while 문의 특징

- 진입조건을 미리 검사를 하고 그 진입 조건을 만족시켰을때 반복이 되는 블록, 혹은 명령문을 실행시킴.

while (expression){
  statement
}

에서 컴퓨터 내부적으로 expression 의 값이 0이면 false, 0이 아닌 값이면 while문을 수행하는 것이다. (0 아닌 값을 주는 여러 트릭으로 이어진다고 함)
