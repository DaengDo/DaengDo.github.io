# 다양한 정수형들

| 정수 자료형                            | 최소 크기(Byte) |                       값의 범위                        | 형식 지정자Format Specifier |
| -------------------------------------- | :-------------: | :----------------------------------------------------: | :-------------------------: |
| _signed_ **char**                      |        1        |                       -128 ~ 127                       |      hhd 또는 c (문자)      |
| _unsigned_ **char**                    |        1        |                        0 ~ 255                         |      hhd 또는 c (문자)      |
| _signed_ **short** int                 |        2        |                    -32,768 ~ 32,767                    |             hd              |
| _unsigned_ **shot** int                |        2        |                       0 ~ 16,383                       |             hu              |
| **_signed_** int 또는 _signed_ **int** |    2 또는 4     |  -32,768 ~ 32,767 or -2,147,483,648 ~ -2,147,483,647   |          d 또는 i           |
| **_unsigned_** int                     |    2 또는 4     |            0 ~ 65,535 or 0 ~ 4,294,967,295             |              u              |
| **_long_** int                         |        4        |            -2,147,483,648 ~ -2,147,483,647             |             ld              |
| **_unsigned_ long** int                |        4        |                   0 ~ 4,294,967,295                    |             lu              |
| **long long** int                      |        8        | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 |             lld             |
| **_unsigned long long int_**           |        8        |             0 ~ 18,446,744,073,709,551,615             |             llu             |

- char 는 문자를 다룰때 더 많이 쓰지만, 가장 작은 자료형으로도 많이 쓴다.
- 다양한 자료형을 활용해 메모리를 효율적으로 써야한다.
- int형 2또는 4 바이트인 이유는 전통적인 이유가 크다.

  - 과거 전자 부품들이 발달하지 않았던 시기에는 2바이트를 사용하는 경우가 많았다고 한다. 그러나 현대에는 int 가 자료형 표준으로 많이 이용되고, 기술이 발달함에 따라 int를 4바이트로 올렸다고 한다. (교수님조차도 2바이트 int를 쓰는 하드웨어를 본적 없으시다고 함.)
  - long은 int가 2byte일 때 int보다 커서 롱이라고 붙였으나 현대에는 int가 4byte 이기 때문에 잘 쓰지 않는다고 한다.(그러고보니 자바와는 크기가 다르네?)
  - 이렇듯이 하드웨어마다 자료형의 크기가 다를 수 있으므로 sizeof() 연산자를 활용해 보는것이 좋다.

- 형식 지정자는 C언어에서만 필수 요소이다.(현대적인 언어에서는 쓰지 않아도 됨)
- 알맞은 형식 지정자를 써서 overflow 가 일어나지 않게해야한다.
