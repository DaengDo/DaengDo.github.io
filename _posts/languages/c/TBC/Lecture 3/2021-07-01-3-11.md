# 부동 소수점형

## 과학적 표기법 (Scientific notations)

### **m X 10^n**

m: significand<br>n: exponent

- 과학이나 공학에서 큰 숫자를 다루기 편하게 하기위해서 사용하는 방식
- 부동 소수점을 사용한다. (정수와 실수 다루는 수업에서 다뤘다.)

$$123.45 = 12345 \times 10^-2 = 1.2345 \times 10^2$$

- 유효숫자 : 얻어진 측정값의 숫자 중 어디까지가 의미있는지를 나타내는 척도
  - 유효 숫자 5개 : $$1.2345 \times 10^2$$
  - 유효 숫자 6개 : $$1.23450 \times 10^2$$

Modified normalized form : $$1.0 \leq m \leq 10$$ <br>
True normalized form : $$0.1 \leq m \leq 1.0$$ <br>
**Normalized significand** : $$1.xxxx \times 2^x$$
Normaed significand : 0.xxxx
<br><br><br>
부동 소수점(floating point)을 쓸 때에는 정해진 비트를 나눠서 쓰기 때문에 그 이상의 값을 표현하려할때 문제가 생긴다.
<br>
4 바이트 정수 범위 : 대략 $$-2.14 \times 10^9 ~ 2.14 \times 10^9$$
<br>
4 바이트 부동소수점수 범위 : 대략 $$-3.4 \times 10^38 ~ 3.4 \times 10^38
<br>

<center> 10진수 기준 유효숫자 6개만 보장된다. </center>
