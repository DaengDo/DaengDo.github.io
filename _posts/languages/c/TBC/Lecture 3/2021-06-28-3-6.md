# 수업중에 나온 키워드 메모

- sizeof();
  - byte 단위로 메모리 사이즈를 알려주는 연산자<br>ex ) 1 이 출력되면 1byte 크기라는 의미
- unsigned int
  - 0과 양수만을 갖는 int 자료형 (0 부터 2^32 - 1)
- UINT_MAX
  - unsigned int 의 max 값<br>(unsigned 의 min 값은 모조리 0이므로 예약어가 없다.)
- INT_MAX
  - int 의 max 값
- INT_MIN
  - int 의 min 값
