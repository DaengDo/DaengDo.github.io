# A summary of printf/scanf format specifiers

## available `printf`, `scanf` format specifiers:

<br>

| format specifier | explanation                                 |
| ---------------- | :------------------------------------------ |
| %c               | character                                   |
| %s               | string                                      |
| %d               | decimal (integer) number (base10)           |
| %e               | exponential floating-point number           |
| %f               | floating-point number (scientific notation) |
| %i               | integer (base10)                            |
| %o               | octal number                                |
| %s               | a stringof characters                       |
| %u               | unsigned decimal (integer)number            |
| %zu              | size_t                                      |
| %x               | number in hexadecimal(base16)               |
| %a               | 16 진수로 출력                              |
| %%               | print a percent sign                        |
| \%               | print a percent sign                        |
| %hhi             | char변수를 숫자로 입출력 할 때              |

<br><br>

- 출력 형식 지정자 갯수와 실제로 넣는 변수 갯수가 같아야 한다.<br>printf("`%d`, `%d`, `%d`", `변수1`, `변수2`, `변수3`)
- 고전적인 C 프로그래밍 스타일은 변수선언을 코드 위쪽에 몰아놓는다.<br> + 현대적으로 초기화까지 해주면 더 좋다
