# 변수(variable)와 상수(constants)

- 프로그래머 입장에서 변수는 어떤 값이 저장될 수 있는 공간
- 메모리 공간에 담기는 값을 litertal constant 라고 한다.

`int angel = 1004;`

- 1004 부분이 literal constant

<br>
리터럴 상수 말고도 다른 상수가 있다. 대표적으로 기호적 상수가 있다.

`const int angel = 1004;`

- 기호적 상수인 const 는 한정자,제한자(Qualifier)라는 용어로 분류된다.
- const 로 인해 상수가 된 angel 값은 기호적 상수(Symbolic constant)라고 부른다.
- 문법상 자료형과 함께 const라는 키워드를 같이 사용하면 angel의 값을 바꿀 수 없다.
