# 1. 블록문

- 코드 블록: 0개 이상의 문을 중괄호로 묶은 것
  ```js
  {
    var foo = 1;
  }
  ```
- 제어문이나 함수를 정의할 때 사용한다.
  ```js
  // 제어문
  var x = 1;
  if (x < 10) {
    x++;
  }
  // 함수 선언문
  function sum(a, b) {
    return a + b;
  }
  ```

<BR>

# 2. 조건문

- 주어진 조건식(conditional expression)의 평가 결과에 따라 코드 블록의 실행을 결정한다.

<BR>

## 2-1. if ... else 문

- 논리적 참 또는 거짓에 따라 실행할 코드 블록을 결정한다.
  ```js
  if (조건식) {
    // 조건식이 참으로 평가되면 이 코드 블록 실행
  } else {
    // 조건식이 거짓으로 평가되면 이 코드 블록 실행
  }
  ```
- 조건에 따라 실행될 코드 블록을 늘리고 싶으면 `else if` 문을 사용한다.
  ```js
  if (조건식1) {
    // 조건식1 참이면 실행
  } else if (조건식2) {
    // 조건식2 참이면 실행
  } else {
    // 조건식1, 조건식2 모두 거짓일시 실행
  }
  ```
- 대부분의 `if...else` 문은 삼항 조건 연산자로 바꿔 쓸 수 있다.
  ```js
  var x = 2;
  var result;

  // if ... else 문
  if (x % 2) {
    result = '홀수';
  } else {
    result = '짝수';
  }
  // 삼항 조건 연산자
  result = x % 2 ? '홀수' : '짝수';
  ```
  - 삼항 조건 연산자는 표현식이기 때문에 값으로 활용할 수 있지만, `if...else` 문은 표현식이 아닌 문이다.

<BR>

## 2-2. switch 문

- 주어진 표현식을 평가하여 그 값과 일치하는 표현식을 갖는 case 문으로 실행 흐름을 옮긴다.
  - 만약 일치하는 표현식이 없다면 실행문은 default 문으로 이동한다.
  ```js
  switch (표현식) {
    case 표현식1:
      // 표현식과 표현식1이 일치하면 실행될 문;
      break;
    case 표현식2:
      // 표현식과 표현식2가 일치하면 실행될 문;
      break;
    default:
      // 표현식과 일치하는 case 문이 없을 때 실행될 문;
  }
  ```
- `if...else`문은 논리적 참, 거짓으로 실행될 코드 블록을 결정하고,<br>`switch`문은 다양한 상황에 따라 실행할 코드블록을 결정할 때 사용한다.

<br>

# 3. 반복문

조건식의 평가 결과가 참인 경우 코드 블록을 실행한다. 조건식이 거짓일 때까지 반복한다.

- `for`문은 반복 횟수가 명확할 때 주로 사용한다.
- `while`문은 반복 횟수가 불명확할 때 주로 사용한다.
- `do while`문은 최소 1번의 반복문을 보장하고 싶을 때 사용한다.

<br>

## 3-1. for 문

```js
for (변수 선언문 또는 할당문; 조건식; 증감식) {
  조건식이 참인 경우 반복 실행될 문;
}
```

<br>

## 3-2. while 문

```js
while (조건식) {
  조건식이 참인 경우 반복 실행될 문;
}
```

<BR>

## 3-3. do while 문

```js
do {
  반복 실행될 문;
} while (조건식)
```

<br>

# 4. break 문

- `break`문은 레이블 문, 반복문 또는 `switch`문의 코드 블록을 탈출한다.
- 중첩된 `for`문을 탈출할 때 유용하지만 그 이외의 상황에서는 프로그램의 복잡성을 높혀 사용이 권장되지 않는다.

<BR>

# 5. continue 문

- 반복문의 코드 블록 실행을 현 지점에서 중단하고 반복문의 증감식으로 실행 흐름을 이동한다.
  - `break`문처럼 반복문을 탈출하지않고 다음 반복 흐름으로 이동한다.
