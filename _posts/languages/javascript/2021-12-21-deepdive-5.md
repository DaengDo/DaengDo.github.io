# 1. 값
- 값(value)은 식(표현식, expresstion)이 평가(evaluate)되어 생성된 결과

<BR>

# 2. 리터럴
- 리터럴(literal: 문자 그대로의), 값을 생성하기 위해 미리 약속한 표기법
  - 정수 리터럴, 부동소수점 리터럴, 2진수 리터럴, 배열 리터럴 등등등

<BR>

# 3. 표현식
- 표현식은 값으로 평가될 수 있는 문<BR>표현식이 평가되면 새로운 값을 생성하거나 기존 값을 참조함
- 표현식과 표현식이 평가된 값은 동치

<BR>

# 4. 문
- 선언문, 할당문, 조건문, 반복문 등으로 구분

<BR>

# 5. 세미콜론과 세미콜론 자동 삽입 기능
- ASI (automatic semicolon insertion)
  - ESLint와 TC39(ECMAScript 기술위원회)가 세미콜론 사용을 권장
