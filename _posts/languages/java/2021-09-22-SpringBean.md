Spring에서 Bean은 IoC 컨테이너가 관리하는 Java 객체를 의미한다.

POJO(Plain Old Java Object)로 Spring 을 구성하는 핵심 객체이다.

Spring IoC 컨테이너에 의해 생성 및 관리된다.

Singleton 방식으로 생성된다.
