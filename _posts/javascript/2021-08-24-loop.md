# while

- 문법
  ```js
  while (condition) {
    // 반복문 본문(body)
  }
  ```
  - `condition`(조건) 이 true 인 동안 반복문 본문을 반복한다. 
- 예시
  ```js
  let i = 0;

  while (i < 3) {
    console.log(i);
    i++;
  }
  // 0, 1, 2가 출력된다.
  ```

<BR><BR>

# do...while

- 문법
  ```js
  do {
    // 반복문 본문(body)
  } while (condition);
  ```
  - 본문이 먼저 실행되고, 조건을 확인한 후 조건이 true인 동안 본문이 계속 실행된다.
  - 본문을 최소한 한번이라도 실행하고 싶을 때만 사용한다.
- 예시
  ```js
  let i = 0;
  do {
    console.log(i);
    i++;
  } while (i < 3);
  // 0, 1, 2가 출력된다.
  ```

<BR><BR>

# for(;;)
- 문법
  ```js
  for(begin; condition; step) {
    // 반복문 본문(body)
  }
  ```
  - `begin`: 반복문에 진입할 때 단 한 번 실행된다.
  - `condition`: 반복마다 해당 조건이 확인된다. false면 반복문을 멈춘다.
  - `body`: `condition`이 true일 동안 계속해서 실행된다.
  - `step`: 각 반복의 body가 실행된 이후에 실행된다.
- 예시
  ```js
  for (let i = 0; i < 3; i++) {
    console.log(i);
  }
  // 0, 1, 2가 출력된다.
  ```

<BR><BR>

# for...in

- 문법
  ```js
  for (let key in obj) {
    // 조건문 본문(body)
  }
  ```
  - JS <b>객체</b> 자료형에서 `key`(속성) 값을 모두 선회할때 사용한다.<BR>객체의 속성 개수만큼 반복한다.
  - `key`: `obj` 객체의 key 값
    - `key`(속성명) 변수는 아무 변수명으로 지어도 상관없다.
  - `obj`: 선회할 객체의 변수명
- 예시
  ```js
  let user = {
    name: "DaengDo",
    age: 28,
    isAdmin: true,
  };

  for (let key in user) {
    console.log(key); // name, age, isAdmin
    console.log(user[key]); // DaengDo, 28, true
  }
  ```

# for...of

- 문법
  ```js
  for (let key of arr) {
    // 반복문 본문(body)
  }
  ```
  - JS <b>배열</b>의 요소를 대상으로 반복작업을 할 때 사용한다.
    - 요소의 인덱스는 얻을 수 없고 값만 얻을 수 있다.<BR>요소의 인덱스를 얻고싶으면 for(;;)문을 사용해야한다.
    - 요소의 개수만큼 반복한다.
    - 요소 변수에는 각 반복에 맞는 요소가 대입된다.
- 예시
  ```js
  let fruits = ["사과", "오렌지", "자두"];

  for (let fruit of fruits) {
    console.log(fruit);
  }
  // 사과, 오렌지, 자두
  ```

<BR><BR>

# forEach

(배열의 메서드 중 하나)

- 문법
  ```js
  arr.forEach(function(item, index, array) {
    // 배열의 요소에 무언가함
  });

  // 화살표 함수로 쓰면?
  arr.forEach((item, index, array) => {
    // 처리할 구문
  })
  ```
  - 주어진 함수를 배열 요소 각각에 대해 실행할 수 있게 해준다.
  - `item`: 요소 변수
  - `index`: 인덱스 변수
  - `array`: 해당 배열 변수
- 예시
  ```js
  ["Bilbo", "Gandalf", "Nazgul"].forEach(alert);
  // Bilbo, Gandalf, Nazgul 이 alert창에 뜬다.

  ["Bilbo", "Gandalf", "Nazgul"].forEach((item, index, array) => {
    console.log(`${item} is at index ${index} in ${array}`);
  });
  ```
