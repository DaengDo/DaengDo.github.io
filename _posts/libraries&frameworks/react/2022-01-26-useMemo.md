연산된 값을 재사용하기 위해 사용하는 훅으로 성능 최적화를 위해 사용한다. 

공식문서에 따르면 `useMemo`를 사용하지 않고도 동작할 수 있도록 코드를 작성하고 필요시 적용해 성능을 최적화하는 것이 바람직하다고 한다. 곧 컴파일러 성능을 높혀 이 훅이 필요 없어질 것 같다.

## 기본형태
```js
import { useMemo } from 'react';

const memoizedValue = useMemo(() => computeExpensiveValue(a, b), [a, b]);
```

- 첫 번째 파라미터에는 연산하는 함수를 넣는다
- 두 번째 파라미터에는 배열을 넣는다. 
  - 배열 안의 내용이 바뀌면 등록한 함수가 실행된다.
  - 배열 안의 값이 바뀌지 않으면 이전에 연산한 값(배열 안의 값)을 재사용한다.
