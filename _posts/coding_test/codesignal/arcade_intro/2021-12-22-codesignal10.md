# Task

Given two strings, find the number of common characters between them.

>Example

For `s1 = "aabcc"` and `s2 = "adcaa"`, the output should be<BR>
`solution(s1, s2) = 3`.

<BR><BR><BR>

# My Solution

```js
function solution(s1, s2) {
    let s1Arr = [...s1];
    let s2Arr = [...s2];
    let alphabetArr = [];
    let result = 0;
    
    // a ~ z 는 97부터 122
    for(let i = 97; i<=122; ++i){
        alphabetArr.push(String.fromCharCode(i));
    }
    
    for(let j = 0; j < alphabetArr.length; ++j){
        let s1Result = s1Arr.filter(item => item == alphabetArr[j]).length;
        let s2Result = s2Arr.filter(item => item == alphabetArr[j]).length;
        
        if(s1Result <= s2Result){
            result += s1Result;
        } else {
            result += s2Result;
        }
    }

    return result;
}
```
- 아스키 코드 변환법(자바스크립트)
  - `String.fromCharCode(숫자)`
    - 숫자를 해당하는 문자열로 변환<BR>(숫자 -> 문자)
  - `charCodeAt(문자)`
    - 문자를 해당하는 숫자로 변환<BR>(문자 -> 숫자)

## Feedback

```js
function solution(s1, s2) {
    let count = 0;
    for(let i = 0; i < s1.length; i++){
        if(s2.indexOf(s1[i]) != -1){
            count++;
            s2 = s2.replace(s1[i], '');
        }
    }
    return count;
}
```

- 내 풀이는 a~z 까지 배열을 만들어둔 다음에 입력 받은 문자열이 겹치는 값이 있나 검사했음
- 정규 표현식을 활용해서 `str.replace`로 겹치는 문자를 하나씩 지워가면서 `count`를 늘리는 풀이가 더 깔끔해 보인다.
