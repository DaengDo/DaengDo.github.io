# Task

Ticket numbers usually consist of an even number of digits. A ticket number is considered lucky if the sum of the first half of the digits is equal to the sum of the second half.

Given a ticket number n, determine if it's lucky or not.

정수형 숫자 주어졌을 때 시각적으로 반갈라서 합했을 때 같은 값인지 여부 채크

>Example

For `n = 1230`, the output should be<BR>
`solution(n) = true`

For `n = 239017`, the output should be
`solution(n) = false`.

<BR><BR><BR>

# My Solution

```js

function solution(n) {

    let nToStr = `${n}`;
    let firstHalf = [];
    let secondHalf = [];
    let firstHalfSum = 0;
    let secondHalfSum = 0;
    
    for(let i = 0; i < Math.floor(nToStr.length / 2); ++i){
        firstHalf.push(+nToStr[i]);
    }
    for(let j = nToStr.length - 1; j >= nToStr.length / 2; --j){
        secondHalf.push(+nToStr[j]);
    }
    
    for(let k = 0; k < firstHalf.length; ++k){
        firstHalfSum += firstHalf[k];
    }
    for(let l = 0; l < secondHalf.length; ++l){
        secondHalfSum += secondHalf[l]
    }
    
    if(firstHalfSum == secondHalfSum){
        return true;
    } else {
        return false;
    }

}

```

- 아무튼 복잡도 n 임 (망함)
