# Task

You are given a two-digit integer `n`.<BR>Return the sum of its digits.

>Example

For `n = 29`, the output should be<BR>
`solution(n) = 11.`

<BR><BR><BR>

# My Solution

```js
function solution(n) {
    // 1. js만의 독특한 방법 이용 (문자열 자르기)
    // let n_string = n.toString()
    // let first_digit = n_string[0];
    // let second_digit = n_string[1];
    // return +first_digit + +second_digit;
    
    // 2. 나머지 연산자(%) 활용
    let second_digit = n % 10;
    let first_digit = ((n - second_digit) * 0.1) % 10;
    return first_digit + second_digit;
}
```

## Feedback

첫 번째 숫자를 나눈 후 소숫점 자리를 버리는 방법도 있다.

```js
// 1. Math.floor 사용
let first_digit = Math.floor(n / 10);

// 2. 비트 연산자 | 를 사용
let first_digit = (n / 10) | 0;
```
두번째 방법?
- 비트 연산자 `|`(OR) 를 사용하면 피연산자가 32비트 int 자료형이 되는 것을 이용해<BR>강제 형 변환으로 소숫점 자리를 버리는 트릭이다.
