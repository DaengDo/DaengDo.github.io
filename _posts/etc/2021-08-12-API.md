# API (application programing interface)

#### 운영체제와 응용프로그램 사이의 통신에 사용되는 언어나 메시지 형식을 말한다.

- 프로그램 개발을 위한 인터페이스
- 서로 다른 프로그램에서 요청과 응답을 주고 받을 수 있게 만든 체계이다.

> CRUD

|Create|POST|
|Read|GET|
|Update|PUT(전체), PATCH(일부)|
|Delete|DELETE|
