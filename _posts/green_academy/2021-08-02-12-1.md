# 1. 변수와 상수

## 1-1. 변수

- 데이터를 저장할 때 쓰이는 '이름이 붙은 저장소'이다.
- `let` 키워드를 사용해 변수를 생성한다.
  ```js
  let message;
  
  message = "Hello";
  ```

<BR>

## 1-2. 변수 명명 규칙

- 문자와 숫자, 그리고 기호 `$`와 `_`만 들어갈 수 있다.
- 첫 글자는 숫자가 될 수 없다.
- 대/소문자를 구분한다.
- 관례적으로 [카멜 표기법](../../scrap/Naming-Convention/#1-camel-case-카멜-표기법-낙타-대문자){:target="_blank"}을 따른다.
- 모든 언어를 변수명에 사용할 수 있으나, 영어가 아닌것은 권장하지 않는다.
- 예약어는 사용할 수 없다.

<BR>

## 1-3. 상수

- `const` 키워드를 사용한다.
  ```js
  const myBirthday = '94.04.26';
  ```
- 기억하기 힘든 값을 변수에 할당해 별칭으로 사용하는 경우에는 대문자와 밑줄로 구성된 이름으로 명명하는 관례가 있다.
  ```js
  const COLOR_RED = "#F00";
  const COLOR_GREEN = "#0F0";
  // 색상을 고르고 싶을 때 별칭을 사용한다.
  let color = COLOR_RED;
  alert(color); // #F00
  ```
> 대문자로 상수를 만들어 사용할때의 장점
  - 기억하기 쉽다.
  - 오타를 낼 확률이 낮다.
  - 코드 가독성이 증가한다.

<BR><BR>

# 2. 자료형

JavaScript에는 8개의 기본 자료형이 있다.

<BR>

## 2-1. 숫자형 (Number type)

- 정수 및 부동소수점 숫자를 나타낸다.
- 숫자형에는 일반적인 숫자 외에 `Infinity`, `-Infinity`, `NaN` 같은 '특수 숫자 값(special numeric value)'이 포함된다.
  - `Infinity` : 무한대(∞)를 나타낸다.
    ```js
    alert( Infinity );
    alert( 1 / 0);
    // 어느 숫자든 0으로 나누면 무한대를 얻을 수 있다.
    ```
  - `NaN` : Not a Number의 약자로, 계산 중에 에러가 발생했다는 것을 나타내주는 값이다.
    ```js
    alert( "문자열" / 2); // 문자열을 숫자로 나누면 에러
    ```
    - Nan은 어떤 추가 연산을해도 결국 NaN이 반환된다.
    - JS에서는 말이 안되는 수학 연산을 해도 NaN을 반환하며 연산이 종료될 뿐이다.

<BR>

## 2-2. BigInt

- `2^53 - 1` 보다 큰 값이나 `-(2^53 - 1)` 보다 작은 정수는 '숫자형'을 사용해 나타낼 수 없고, BigInt를 사용해 나타낼 수 있다.
- 정수 리터럴 끝에 `n`을 붙이면 만들 수 있다.
```js
const bigint = 123456789123456789123456789123456789n;
```
- 자주 쓰이지 않는다.

<BR>

## 2-3. 문자형 (String)
JS에서 따옴표를 활용해 문자열을 나타내는 자료형이다.

>따옴표의 3가지 종류
- 큰따옴표: `"Hello"`
- 작은따옴표: `'Hello'`
- 역 따옴표(백틱, backtic): ``` `Hello` ```

- 큰따옴표와 작은따옴표의 차이를 두지 않는다.
- 역 따옴표로 변수나 표현식을 감싼 후 `${...}`안에 넣어주면, 원하는 변수나 표현식을 문자열 중간에 넣을 수 있다.
  ```js
  let name = "JavaScript";

  // 변수를 문자열 중간에 삽입
  alert( `Hello, ${name}\!` ); // Hello, JavaScript!

  // 표현식을 문자열 중간에 삽입
  alert( `the result is ${1 + 2}` ); // the result is 3
  ```
<BR>

## 2-4. 불린형 (Boolean)
불린형(논리타입)은 `true`와 `false` 두 가지 값 밖에 없는 자료형이다.

<BR>

## 2-5. 'null' 값

- `null` 값은 어느 자료형에도 속하지 않는 값이다.
- `null` 값은 오로지 `null` 값만 포함하는 별도의 자료형을 사용한다.
- 자바스크립트에선 `null`을 '존재하지 않는(nothing)' 값, '비어 있는(empty)' 값, '알 수 없는(unknown)' 값을 나타내는 데 사용한다.

<BR>

## 2-6. 'undefined' 값

- `undefined` 값 또한 `null` 값처럼 자신만의 자료형을 형성한다.
- '값이 할당되지 않은 상태'를 나타낼 때 사용한다.
- 변수는 선언했지만, 값을 할당하지 않았다면 해당 변수에 `undefined`가 자동으로 할당된다.

<BR>

## 2-7. 객체와 심볼

수업도중 배우지 않았다. 나중에 배워서 내용을 채워놓을 생각이다.

<BR>

## tips) typeof 연산자
(자료형 아님, 연산자임)

- `typeof` 연산자는 인수의 자료형을 반환한다.
- 자료형을 알아내고자 할 때 유용하다.
```js
typeof undefined // "undefined"
typeof 0 // "number"
typeof 10n // "bigint"
typeof true // "boolean"
typeof "DaengDo" // "string"
```

# 3. alert, prompt, confirm

브라우저 환경에서 사용되는 최소한의 사용자 인터페이스 기능들이다.

## 3-1. alert

- 확인 버튼을 누를 때 까지 메시지를 보여주는 창을 띄우는 `alert` 함수이다.

<BR>

## 3-2. prompt

```js
result = prompt(title, [default]);

// title : 사용자에게 보여줄 문자열
// default : 입력 필드의 초기값(선택값)
```
- 두 개의 인수를 받고, `prompt`함수가 실행되면 '텍스트 메시지'와 '입력 필드(input field)', '확인' 및 '취소'버튼이 있는 모달 창을 띄워준다.

- alert, prompt 활용 예시
  ```js
  let name = prompt("이름을 입력해주세요!", "이름?");

  alert(`안녕하세요, ${name}님!`);
  ```

<BR>

## 3-3. confirm
```js
result = confirm(question);
```
- `confirm` 함수는 'question'과 '확인' 및 '취소' 버튼이 있는 모달 창을 보여준다.
- 사용자가 확인 버튼을 누르면 `true`,<BR>그 외에는 `false`를 반환한다.
```js
let isAdmin = confirm("관리자 권한으로 실행하시겠습니까?");

alert( isAdmin ); // 확인 버튼을 눌렀다면 true가 출력된다.
```
