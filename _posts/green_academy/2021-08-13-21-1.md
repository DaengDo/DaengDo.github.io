# 1. 자주 사용되는 DOM 이벤트

> 마우스 이벤트
- `click` : 요소 위에서 마우스 왼쪽 버튼(터치스크린 장치에서의 탭)을 눌렀을 때 발생한다.
- `contextmenu` : 요소 위에서 마우스 오른쪽 버튼을 눌렀을 때 발생한다.
- `mouseover`, `mouseout` : 마우스 커서를 요소 위로 움직였을 때, 커서가 요소 밖으로 움직였을 때 발생한다.
- `mousedown`, `mouseup` : 요소 위에서 왼쪽 버튼을 누르고 있을 때, 마우스 버튼을 뗄 때 발생한다.
- `mousemove` : 마우스를 움직일 때 발생한다.

> 키보드 이벤트
- `keydown`, `keyup` : 사용자가 키보드 버튼을 누르거나 뗄 때 발생한다.

> 폼 요소 이벤트
- `submit` : 사용자가 `<form>` 을 제출할 때 발생한다.
- `focus` : 사용자가 `<input>` 과 같은 요소에 포커스 할 때 발생한다.

> CSS 이벤트
- `transitionend` - CSS 애니메이션이 종료되었을 때 발생한다.

> 문서 이벤트
- `DOMContentLoaded` : HTML이 전부 로드 및 처리되어 DOM 생성이 완료 되었을 때 발생한다.

# 2. addEventListener

DOM 요소에 이벤트를 추가할 수 있는 키워드이다. 문법은 다음과 같다.
```js
element.addEventListener(event, handler, [options]);
```
- `event` : 이벤트 이름
- `handler` : 핸들러 함수(이벤트 시 수행할 함수)
- `options` : 특정 프로퍼티를 갖는 객체일 경우 (추가 공부필요 수업때 안함)
  - `once` : true 면 이벤트가 트리거 될 때 이벤트리스너가 자동으로 삭제된다.
  - `capture` : 어느 단계에서 이벤트를 다뤄야 하는지를 알려주는 프로퍼티이다.

<BR>

- 핸들러 삭제는 `removeEventListner` 로 한다.
```js
element.removeEventListener(event, handler, [options]);
```
  - 핸들러를 삭제하려면 핸들러 할당 시 사용한 함수를 그대로 전달해줘야한다.(익명 함수 사용시 삭제 불가능하다.)

<BR><BR>

# 3. 이벤트 객체

이벤트가 발생하면 브라우저는 '이벤트 객체(event object)'라는 것을 만든다.

이벤트 객체로부터 이벤트 정보를 추출해올 수 있다.

```html
<html>
  <body>
    <button id="elem">클릭해 주세요.</button>
    <script>
      const elem = document.querySelector("#elem");

      // 이벤트가 실행될 때 event 객체를 매개변수로 넘겨줌
      const click = (e) => {
        console.log(e);
        console.log(e.target);
        console.log(e.type);
        console.log(e.clientX);
        console.log(e.clientY);
      }

      elem.addEventListener("click", click);
    </script>
  </body>
</html>
```
