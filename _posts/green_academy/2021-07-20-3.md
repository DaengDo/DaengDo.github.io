# 1. 글자 태그

<BR><BR>

`<b></b>`, `<strong></strong>`

<div class="notice--primary" markdown="1">
```
<b>Bold</b>, <strong>Strong</strong>
``` 
 - <b>Bold</b>, <strong>Strong</strong>
</div>


- 글자를 굵게(bold) 표현해준다. strong은 중요한 의미가 추가된 b의 시맨틱 태그이다.<BR>시각장애인 리더기에서 강한 어조를 표현할 수 있다.
  - semantic tag : 기계가 이해할 수 있게 html 문서를 정보화하는 것


_시멘틱 태그 용도 - 기계가 잘 이해할 수 있게 html 문서를 만드는것_

<BR><BR>

`<i>`, `<em> `

<div class="notice--primary" markdown="1">
```
<i>Italic</i>, <em>Emphasis</em>
``` 
 - <i>Italic</i>, <em>Emphasis</em>
</div>

- 글자를 기울여준다. 시맨틱 태그 em(emphasis)으로 강조 표현이 가능하다.

<BR>

`<small>`

<div class="notice--primary" markdown="1">
```
母<small>어미 모</small>
``` 
 - 母<small>어미 모</small>
</div>

- 글자를 작게 표현해준다. (일본어나 한자어를 표기할때 이용할 것 같다?)

<BR>

`<sup>`, `<sub>`

<div class="notice--primary" markdown="1">
```
Super subscript<sup>1</sup>, Subscript<sub>2</sub>
``` 
 - Super subscript<sup>1</sup>, Subscript<sub>2</sub>
</div>


- 글자를 위/아래쪽에 출력해 첨자를 표현할 수 있다. (갓무위키의 주석)

<BR>

`<u>`, `<ins>`



```
<u>Underline</u>, <ins>Insert</ins>
``` 

- <u>Underline</u>, <ins>Insert</ins>
- 글자에 밑줄을 넣는다.<BR>insert 태그는 datetime 속성값을 이용하여 내용의 추가가 언제 이루어졌는지를 알려줄 수 있는 시맨틱 태그다.

```
<ins datetime="2021-07-20">2021년7월21일 추가 텍스트입니다.</ins>
```
- <ins datetime="2021-07-20">2021년7월21일 추가 텍스트입니다.</ins>

<BR>

`<s>`, `<del>`
<div class="notice--primary" markdown="1">
```
<s>Strikethrough</s>
<del>Delete</del>
``` 

- <s>Strikethrough</s><del>Delete</del>

</div>

- 글자에 중간선을 넣는다. del이 s의 시맨틱 태그다.

# 2. 테이블 태그

> 테이블 태그

- 구조

```
<table>
  <thead>
    <tr>
      <th>Table Header1</th>
      <th>Table Header2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Table Data1</td>
      <td>Table Data2</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Table Footer1</td>
      <td>Table Footer2</td>
    </tr>
  </tfoot>
</table>
```

- `<table></table>` : 가로/세로가 있는 표 형식을 표현하는 태그
- `<thead></thead>` : table head, 테이블의 해더 부분 시맨틱 태그
- `<th></th>` : table header, 테이블의 해더
- `<tr></tr>` : table row, 테이블의 행/가로축
- `<td></td>` : table data, 테이블의 셀 안에 들어가는 데이터
- `<tbody></tbody>` : table body, 테이블의 몸체 부분 시맨틱 태그
- `<tfooter></tfooter>`: table footer, 테이블의 바닥 부분 시맨틱 태그

## tbody 태그 자동생성 (개발자도구)

```
<table>
  <tbody>
    <tr>
        <td></td>
    </tr>
  <tbody>
<table>
```

> tbody 태그는 사용하지않더라도 크로니움에서 자체적으로 부여한다.
> 따라서 css나 js 를 쓸때 코드가 문제 될 수 있으니 만들때 tbody 태그를 넣는것이 권장된다.

- tips
  - 테이블은 pc를 위한 레이아웃이다. 모바일 인터페이스에서는 온전히 출력이 안될 가능성이 있다. 화면이 작아지면 카드 레이아웃으로 변경되게끔 하는것이 권장된다.

# 3. 공간 분할 태그

- `<div>` div(division)

- 블록 형식 태그
