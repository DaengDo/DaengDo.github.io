# 1. while 반복문

`while` 반복문의 문법은 다음과 같다.
```js
while (조건식,condition) {
  // 코드
  // '반복문 본문(body)'
}
```

- 조건식(condition)이 true 로 판별되는 동안 반복문의 본문이 실행된다.

```js
let i = 0;

while ( i < 3 ) {
  console.log( i );
  i++;
}
```
- 본문이 한 줄이면 대괄호를 쓰지 않아도 된다.

```js
let i = 0;

while ( i < 3 ) alert( i++ )
```

<BR>

# 2. do while 반복문

`do ... while` 반복문을 사용하면 조건식을 반복문 본문 아래로 옮길 수 있다.
```js
do {
  // body
} while ( 조건식, condition )
```
조건을 나중에 확인하니 본문이 최소 한번은 실행이 된다. 

<BR><BR>

# 3. for 반복문

`for` 반복문의 문법은 다음과 같다.
```js
for ( begin; condition; step ) {
  // 반복문 본문(body)
}
/* 예시 */
for ( let i = 0; i < 5; ++i ) {
  console.log( i );
}
```
<BR>

|구성요소| 예시 | 설명 |
|---|---|---|
| begin | let i = 0 | 반복문에 진입할 때 단 한 번 실행된다.|
| condition | i < 5 | 반복마다 해당 조건을 확인한다. false로 판별되면 해당 반복문을 종료한다. |
| step | ++i | 각 반복의 body가 실행된 이후에 실행된다. |
| body | console.log( i ) | 조건문이 true일 동안 계속해서 실행된다. |


> 실행 순서<BR>begin → condition → body → step<BR>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;→ condition → body → step<BR>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;→ condition → body → step <BR> → condition이 false일 경우 반복문 종료

- 구성요소의 생략이 가능하다.

```js
/* begin 생략 */
let i = 0;
for ( ; i < 3; i++ ) {
  console.log( i );
}

/* step 생략 */
let i = 0;
for ( ; i < 3; ) {
  console.log( i++ )
} // while ( i < 3 ) 와 같다.

/* 모든 구성요소 생략 */
for ( ;; ) {
  // 본문이 끊임 없이 실행됨
}// 무한 루프
```

<BR><BR>

# 3. 반복문 빠져나오기 ( break )

반복문의 조건 불린값이 false가 되거나 <BR>`break`를 사용하면 원하는 때에 반복문을 빠져나올 수 있다.

```js
let sum = 0;

while ( true ) {
  let value = +prompt( "숫자를 입력하세요.", "" )
  if ( !value ) break; // !value가 true값이 되면 반복문 탈출
  sum += value;
}

alert( "합계 : " + sum );
```
위의 예시에서 !value가 true 값이 되는건 공백(`""`)을 제출하거나 취소(`null`)를 눌러서 형 변환이 0으로 되었을 경우이다.

<BR><BR>

# 4. 다음 반복으로 넘어가기 ( continue )

`continue` 지시자는 현재 실행 중인 반복문을 멈추고 다음 반복을 수행시킨다.<BR>아래 예시는 `continue` 지시자를 활용해 짝수만을 출력하는 예시이다.

```js
for ( let num = 1; num < 10; num++ ) {
  if ( num % 2 == 1 ) continue;
  console.log( num );
}
```

<br><BR>

# 5. 레이블

중첩 반복문을 한 번에 빠져나와야 하는 경우 레이블을 이용한다.
```js
labelName: for(...) {
  for(...) {
    for(...) {
      /* 중첩 반복문을 나가고싶다면! */
      break labelName;
    }
  }
}
```
- `break` 지시자 뿐만 아니라 `continue` 지시자 또한 사용이 가능하다.
- `break`와 `continue`는 반복문 안에서만 사용할 수 있고, 레이블은 반드시 `break`나 `continue` 지시자 위에 있어야 한다.
