# 1. QuerySelectorAll

`elem.querySelectorAll(CSS선택자)` 는 elem의 자식 요소 중 주어진 CSS선택자에 대응하는 요소 모두를 반환한다.

```html
<ul>
  <li>1-1</li>
  <li>1-2</li>
</ul>
<ul>
  <li>2-1</li>
  <li>2-2</li>
</ul>
<script>
  let elements = document.querySelectorAll('ul > li:last-child');

  for (let elem of elements) {
    alert(elem.innerHTML); // "1-2", "2-2"
  }
</script>
```

<BR>

# 2. QuerySelector

`elem.querySelector(CSS선택자)` 는 주어진 CSS 선택자에 대응하는 요소 중 첫 번째 요소를 반환한다.

- `elem.querySelectorAll(css)[0]`과 동일하나,<BR>`querySelectorAll`은 모든 요소를 검색해 첫 번째 요소만을 반환하고, `querySelector`는 해당하는 요소를 찾으면 검색을 멈춘다는 차이점이 있다. (속도의 문제)
