# 1. React ?

React는 페이스북에서 만든 자바스크립트 UI 라이브러리이다.<BR>React를 활용하면 컴포넌트(사용자 설정 태그)를 중심으로 현대적인 웹 애플리케이션 UI를 만들 수 있다.

<BR><BR>

# 2. 개발환경 구축

Create-React-App 을 사용한 React 개발환경 구축 방법을 배웠다.<BR>
Create-React-App을 설치하기 위해서는 npm이 필요한데 npm은 Node.js로 만들어진 여러 앱을 명령어 환경에서 손쉽게 설치할 수 있게 도와주는 도구이다.<BR>(npm은 Node.js를 위한 일종의 앱스토어 같은 것) 

따라서 Node.js (npm) -> Create-React-App 순서대로 구축을 해서 React 개발 환경을 구축하면 된다.

<BR>

## 2-1. Node.js (npm)

https://nodejs.org/ko/

[공식 홈페이지](https://nodejs.org/ko/){:target=_blank}에서 Node.js를 다운로드 받고 설치를 진행한다.

Node.js는 JS 빌드와 JS 라이브러리 관리를 도와주는 프로그램이다.

cmd 창에<BR>
`node --version`<BR>
`npm --version`<BR>
을 입력하여 잘 설치되었는지 확인해보면 된다.

<BR>

## 2-2. npm 이용 방법

cmd 창에서 명령어로 원하는 모듈(특정 기능을 하는 JS 소스코드들의 모음)을 받는 방법은 다음과 같다.

- `npm install 모듈명(라이브러리이름)`
  - 해당 모듈을 npm 저장소(npmjs)에서 받아와서 설치한다.
- `-g` (global, 전역)를 기입하면 어디서나 해당 모듈을 사용할 수 있다.<BR>`-g` 옵션이 없다면 현재 폴더에 서만 이용 가능한 모듈이 설치된다.
- `npx` 는 npm 명령어에 실행 명령어를 합친 것으로, 모듈을 설치하고 실행하라는 명령어이다.

<BR>

## 2-3. Create-React-App

`npx install -g create-react-app` 을 cmd창에 입력해 설치한다.
