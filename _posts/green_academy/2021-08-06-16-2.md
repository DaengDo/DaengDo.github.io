## 1. 참조에 의한 객체 복사

객체는 원시 타입(문자열, 숫자, 불린 값 등)과 다르게 변수에 그대로 저장되지 않고,<BR>객체가 저장되어 있는 메모리 주소가 저장된다.
```js
let primaryType = 1;        // Number 자료형 저장
primaryType = "abcdef"      // String 자료형 저장
console.log(primaryType);   // abcdef


let obj1 = {
  name: "DaengDo",
  age: 28,
}
let obj2 = obj1
obj2.isMale = true;         // obj2에서 property를 추가

console.log(obj1.isMale);   // true, obj1에서도 추가된 모습을 나타낸다.
```

> 참조에 의한 비교

- 객체 비교시 동등 연산자 `==` 와 일치 연산자 `===`는 동일하게 작동한다.
- 비교 시 피연산자인 두 객체가 동일한 객체인 경우 true를 반환한다. (아래 예시)
  ```js
  let a = {};
  let b = a;

  console.log(a == b);        // true
  console.log(a === b);       // true
  ```

- 참조가 같지 않으면 독립된 객체로 인식한다. (아래 예시)
  ```js
  let objA = {};
  let objB = {};

  console.log(objA == objB);  // false
  ```

## 2. 객체 복사, 병합 (object.assign)

객체를 복사하는 방법

- 새로운 객체를 만든 다음 기존 객체의 프로퍼티들을 순회해 복사하는 방법 (아래 예시)
  ```js
  let user = {
    name: DaengDo,
    age: 28,
  };
  let clone = {};     // 복사본이 될 빈 객체

  // 빈 객체에 복사할 객체의 property 전부를 복사
  for (let key in user) {
    clone[key] = user[key];
  }
  ```

  <BR>

- `Object.assign` 메서드를 사용해 여러 객체를 하나로 병합하는 방법 (아래 예시)
  ```js
  // 메서드 문법은 다음과 같다.
  Object.assign(목표객체, [합칠객체1, 합칠객체2])

  // 예시
  let user = {name: "DaengDo"};
  let permission1 = {canView: true};
  let permission2 = {canEdit: true};

  // permission1과 2의 property를 user로 복사한다.
  Object.assign(user, permission1, permission2);

  console.log(user); 
  // user = {name: "DaengDo", canView: true, canEdit:true};
  ```
  - 목표 객체에 동일한 이름을 가진 프로퍼티가 있는 경우 기존 값이 덮어씌워진다.
    ```js
    let user = {name: "DaengDo"};

    Object.assign(user, {name: "Kim"});

    console.log(user.name);  // Kim
    ```
  - `Object.assign` 을 사용하면 반복문 없이도 간단하게 객체를 복사할 수 있다.
    ```js
    let user = {
      name: "DaengDo",
      age: 27,
    }

    let clone = Object.assign({}, user);
    ```
- spread(나열)`...` 연산자를 활용한 object literal 표현식
  ```js
  // let clone = {}; -> 새로운 객체 생성
  // {...user} -> 새로운 객체에 모든 속성과 값을 나열
  // let clone = {name: user.name, age: user.age};

  let user = {
    name: "DaengDo",
    age: 28,
  };
  let clone = {...user};  // 독립적인 복제본 객체 생성
  ```
  - `...` 연산자를 이용한 복사는 '얕은' 복사를 수행하며, 복사 대상에 객체가 있는 경우에는 객체 자체가 복사되지않고 원본 값을 참조한다.
    ```js
    let user = {
      name: "DaengDo",
      permission : {
        canView: true,
        canEdit: true,
      },
      age: 28,
    };

    let clone = {...user};

    user.permission.canEdit = false;

    console.log(clone.permission.canEdit); // false
    ```
