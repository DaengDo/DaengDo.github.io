# 1. 전개 구문(spread operator)

`[...logs]`

배열의 요소를 콤마로 구분지어 문자열로 반환해주는 연산자이다.

```js
let arr1 = [0, 1, 2];
let arr2 = [...arr1];
console.log(arr2); // [0, 1, 2]

console.log([-1, ...arr1, 3]) // [-1, 0, 1, 2, 3]
```

React에서 state 조작함수로 주로 쓰여서 익혔다. 다만, 아래의 리엑트 라이브러리인 immer를 사용하면 state에도 일반 JS 객체나 배열의 메서드를 사용할 수 있다.

<BR><BR>

# 2. immer

- 리엑트를 위한 라이브러리
- state를 불변성을 지키면서 일반 JS 객체/배열 다루듯 다룰 수 있게 해준다.
  - `current state` -> `draft state` -> `next state`<BR>`current state`를 조작하는 것처럼 보이나 `dratf state`를 조작해 `next state`를 조작하는 방식이다.
  ```ts
  import produce from "immer"

  produce(currentState, recipe: (draftState) => void): nextState

  // 예시
  produce((state) => {
    state.unshift(todo); // unshift메서드가 사용가능
  })
  ```
- `npm install immer`
- https://immerjs.github.io/immer/


<BR><BR>

# 3. state 변경

### spread operator 사용시
1. 현재 state 존재
2. 새로운 state 생성
3. 현재 state 변경 함수에 새로운 state를 매개변수로 삽입
4. 변경된 state 탄생

### immer 사용시
1. 현재 state 존재
2. immer가 만들어준 변경될 state를 조작(splice, push 등)
3. state 변경함수에 변경될 state를 넣음
4. 변경된 state 탄생
