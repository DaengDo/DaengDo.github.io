# 2. HTML 기초 지식

- HTML,CSS의 업무가 점점 프론트엔드 개발자의 영역으로 들어옴에 따라 기술 중요도가 높아지고 있음
- 하이퍼 텍스트는 텍스트의 링크를 달아 논문을 게재하기 위한 용도로 탄생
- HTML5 로 발전하며 추가 확장자 없이 다양한 기능 제공
- 계층구조를 갖고 있음(상/하위, 부모자식)
- HTML 문법 구성

- `<!-(주석)-->`

## sementic tag
- 태그의 기능을 분별하는 의미상의 태그<br>검색 엔진 노출 최적화(SEO, Search Engine Optimization)를 위해 이용
- header, nav, aside, section, article, footer 이 존재


## 수업중 배운 태그

- `<html>` <br>html 문서의 시작과 끝
- `<head>`<br>문서의 머리를 나타내는 태그<br>브라우저 화면에 직접적으로 보이지 않으며, 숨은 데이터를 정의하는 태그들이 들어감
- `<body>`<br>문서의 몸통을 나타내는 태그<br>브라우저 화면에 보이는 것들이 주로 들어감
- `<h1~23456>`<br>header(제목)태그, 총 6가지 크기가 존재
- `<p>`<br>paragraph, 문단을 표현
- `<ul>`,`<ol>`,`<li>`<br>unordered list, ordered list를 표현
- `<br>` <br>띄어 쓰기
- `<hr>` <br>줄 긋기
