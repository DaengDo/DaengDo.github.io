# 1. 요소 추가/제거 메서드

앞 수업에서 배운 메서드
- `push(...items)` : 맨 끝에 요소 추가
- `pop()` : 맨 끝에 요소 제거
- `shift()` : 맨 앞에 요소 삭제
- `unshift(...items)` : 맨 앞에 요소 추가

<BR>

## 1-1. splice

- `arr.splice(index, deleteCount, elem1, ..., elemN)`<BR><BR>index 부터 deleteCount까지 요소를 제거하고 elem1, ..., elemN 을 추가해 반환한다.<BR>
  ```js
  let arr = ["I", "study", "JavaScript"];

  arr.splice(1, 1, "love"); // 인덱스 1부터 요소 한 개를 제거하고 "love" 추가

  console.log(arr); // ["I", "love", "JavaScript"]
  ```

<BR>

## 1-2. slice

- `arr.slice(start, end)`

start부터 end 바로 앞 까지 요소를 복사해 새로운 배열을 만든다.<BR>start와 end를 음수로 지정해주면 배열 끝에서부터의 요소 개수를 의미한다.

```js
let arr = ["t", "e", "s", "t"];

console.log(arr.slice(1, 3));   // e,s

console.log(arr.slice(-2)); // s,t
```

- `arr.slice()`는 인수를 하나도 넘기지 않고 호출하여 arr의 복사본을 만들 수 있다.<BR>(기존의 배열을 건드리지 않으면서 배열을 조작해 새로운 배열을 만듦)

<BR>

## 1-3. concat (concatenation, 합치기)

- `arr.concat(arg1, arg2...)`

배열의 모든 요소를 복사하고 arg(인수)를 추가해 새로운 배열을 만든 후 이를 반환한다. arg가 배열이면 이 배열의 인수를 기존 배열에 더해준다.

```js
let arr = [1, 2];

// 새로운 배열 생성
let concatedArr = arr.concat([3, 4])
console.log(concatedArr);                 // 1,2,3,4

// arg가 배열인 경우
console.log(arr.concat([3, 4], [5, 6]));  // 1,2,3,4,5,6
// arg가 인수인 경우와 배열인 경우
console.log(arr.concat([3, 4], 5, 6));    // 1,2,3,4,5,6
```

# 2. forEach (배열 전체 순회)

`arr.forEach(function(item, index, array) {});`<BR>
모든 요소에 함수를 호출한다.

```js
// 모든 요소를 얼럿창을 통해 출력한다.
["Bilbo", "Gandalf", "Nazgul"].forEach(alert);

// 인덱스 정보까지 더해서 출력해주는 좀 더 정교한 코드
["Bilbo", "Gandalf", "Nazgul"].forEach((item, index, array) => {
  alert(`${item} is at index ${index} in ${array}`);
});
```

# 3. 배열 탐색

## 3-1. indexOf / lastIndexOf

`indexOf/lastIndexOf(item, pos)`
- `pos`부터 원하는 `item`을 찾는다. 찾게 되면 해당 요소의 인덱스를, 아니면 -1을 반환한다.

<BR>

## 3-2. includes

`includes(value)`
- 배열에 `value`가 있으면 `true`를, 그렇지 않으면 `false`를 반환한다.

<BR>

## 3-3. find / filter

`find / filter(func)`

- `func`의 반환 값을 `true`로 만드는 첫 번째 / 전체 요소를 반환한다.

```js
/* find */
let result = arr.find(function(item, index, array) {
  // true가 반환되면 반복이 멈추고 해당 요소를 반환한다.
  // 조건에 해당하는 요소가 없으면 undefined를 반환한다.
});
```

```js
/* filter */
let results = arr.filter(function(item, index, array) {
  // 조건을 충족하는 요소는 results에 순차적으로 더해진다.
  // 조건을 충족하는 요소가 하나도 없으면 빈 배열이 반환된다.
});

```

<BR>

## 3-4. findIndex

`findIndex`

- `findIndex`는 `find`와 유사하나, 요소 대신 인덱스를 반환한다. 반환할 요소가 없으면 `-1`을 반환한다.

# 5. Array.isArray (배열 여부 확인)

자바스크립트에서 배열은 독립된 자료형으로 취급되지 않고 객체형에 속한다.<BR>따라서 `typeof`로는 일반 객체와 배열을 구분할 수 없다.

`Array.isArray(value)` 메서드를 사용해 `value`가 배열이라면 `true`를 배열이 아니라면 `false`를 반환한다.
```js
let obj = {};
let arr = [];

console.log(Array.isArray(obj)); // false

console.log(Array.isArray(arr)); // true
```
