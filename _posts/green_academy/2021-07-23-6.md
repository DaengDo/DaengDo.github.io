# 1. pseudo (의사 클래스)

- `:first-child` 
  - 선택자와 같은 레벨의 요소들 중 첫번째 요소를 선택한다.
- `:last-child`
  - 선택자와 같은 레벨의 요소들 중 마지막 요소를 선택한다.
- `:nth-child`
  - 선택자와 같은 레벨의 요소들 중 수열식에 맞는 요소를 선택한다.

<BR><BR>

# 2. pseudo element (의사 요소)

> pseudo element(가짜,의사 요소) : element는 아니지만 element와 비슷하다.

- `::fist-letter`
  - 해당 선택 요소 컨텐트의 첫번째 글자에 적용한다.
- `::first-line`
  - 해당 선택 요소 컨텐트의 첫번째 줄에 적용한다.
- `::before`
  - 태그 앞에 위치하는 공간을 선택한다.<BR>시작태그와 컨텐트 사이의 before영역에 스타일을 넣는다.
- `::after`
  - 태그 뒤에 위치하는 공간을 선택한다.<BR>끝태그와 컨텐트 사이의 after 영역에 스타일을 넣는다.
- `::selection`
  - 사용자가 드래그한 글자를 선택한다.

<BR><BR>

# 3. properties and values

- `counter-increment: 변수명`
  - 증가연산자로, 다른 property의 value로 쓰인다.<BR> counter(변수명) 으로 활용한다.
- `attr(data-count)`
  - attr()의 하나다.
  - 나중에 JS로 제어하는 값이라고 한다.
  - 관련자료 모질라 <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/attr()" target="_blank">링크</a>

<BR><BR>

## CSS 적용 우선순위

- 상위 요소의 스타일보다 하위 요소의 스타일이 우선순위가 높다.
- 범위가 넓은 것 보다 좁은 것의 우선순위가 높다. (ex: 태그보다 클래스가 클래스보다 id가 우선 순위가 높다.)
