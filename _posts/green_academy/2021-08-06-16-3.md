## 4. this가 없는 화살표 함수

this context(운영체제 용어)가 다른 경우가 있어서 화살표 함수가 나온거다.
this가 고정이 안되는 경우가 나와서 (window 일때도 잇고 네트워크 일때도잇고) 전역함수로 아예 고정시킨게 화살표 함수
화살표 함수 쓰면 this 고려 안해도됨

context : 멀티쓰레드 없던 시절에 쓰레드 하나로 실행되는걸 분리해서 실행

자바스크립트는 쓰레드가 한개라서 context 로 처리함
