# 1. switch

복수의 `if` 조건문은 `switch` 문으로 바꿀 수 있다.

<BR><BR>

# 2. 문법

`switch`문은 하나 이상의 `case`문으로 구성된다.<BR>대게 `default`문도 있지만, 필수는 아니다.
```js
switch( 변수 ) {
  case 'value1':      // if ( 변수 === 'value1' )
    ...
    break;
  
  case 'value2':      // if ( 변수 === 'value2' )
    ...
    break;
  
  default:
    ...
    break;
}
```
- 변수의 값과 첫 번째 case문의 값 value1 을 일치 비교한 후, 두 번째 case문의 값 value2 와 비교한다. (이하 반복)
- 변수의 값과 case 문의 값이 일치한다면 case 문의 아래의 코드를 실행한다.
- 일치하는 case 문이 없다면 (default 문이 있을 경우에) default 문 아래의 코드가 실행된다.

```js
let test = prompt("엄마 또는 아빠만 입력하세요", "");

switch ( test ) {
  case "엄마":
    alert ( "엄마가 최고!" );
    break;

  case "아빠":
    alert ( "아빠가 최고!" );
    break;

  case null:
    alert ( "선택을 포기하셨군요.." );
    break;

  default:
    alert ( "골라야해!" );
    break;
}
```

<BR><BR>

# 3. 여러 개의 "case" 문 묶기

실행하려는 코드가 같은 `case` 문은 한데 묶을 수 있다.

위의 예시를 수정해보았다.
```js
let test = prompt("엄마 또는 아빠만 입력하세요","");

switch ( test ) {
  case "엄마": // 두 case 문을 묶음
  case "아빠":
    alert ( "최고!" );
    break;

  case null:
    alert ( "선택을 포기하셨군요.." );
    break;

  default:
    alert ( "골라야해!" );
    break;
}
```
`case "엄마"`와 `case "아빠"`는 동일한 메시지를 보여준다.<BR>
`switch/case`문에서 `break`문이 없는 경우엔 조건에 상관없이 다음 `case`문이 실행된다. 그 원리를 활용해 `case`문을 한데 묶은 것이다. 

<BR><BR>

## 주의사항 : 자료형

switch 문은 일치 비교로 조건을 확인한다.

비교하려는 값과 case문 값의 자료형이 일치해야 case 문이 실행되므로 자료형의 확인이 중요하다.

```js
let promptTest = prompt("1이나 2를 입력하세요");

switch ( promptTest ) {
  case "1" :
  case "2" :
    alert ( "문자열 1과 2" );
    break;
  case 1 :
  case 2 :
    alert ( "숫자 1과 2" );
    break;
  default:
    alert ( "error" );
    break;
}
```

prompt 함수는 입력받은 값을 문자열로 바꿔서 출력한다.<BR>따라서 숫자형인 `case 1`과 `case 2`는 절대 실행되지 않는다.
