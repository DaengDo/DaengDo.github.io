# 1. 화살표 함수

- 함수 표현식을 화살표 함수를 이용해 표현할수도있다. (함수 선언문은 불가능)
- 문법은 다음과 같다.

```js
// 함수 표현식
let func = function(arg1, arg2, ...argN) {
  return expression;
};

// 위 코드를 화살표 함수로 표현
let func = (arg1, arg2, ...argN) => expression


// 예시 코드 (덧셈함수)
let sum = (a, b) => {   // 중괄호는 본문이 여러 줄로 구성되어 있음을 알려준다.
  let result = a + b;
  return result;        // 중괄호를 사용했다면, return 지시자로 결과값을 반환해야 한다.
};

alert( sum(1, 2) );     // 3
```
  - `let 변수 = (매개변수) => { ... 구문 }` 의 형태이다.
  - `{...}` 중괄호 구문이 생략되면 해당 구문의 값이 return 값이 된다.
  - 인수가 하나도 없을 때는 괄호를 비워두면 된다. (생략불가능)

```js
let sayHi = () => alert("안녕하세요!");

sayHi();
```
