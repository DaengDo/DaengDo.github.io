# 1. Flex

Flex 속성은 하나의 플렉스 아이템이 자신의 컨테이너가 차지하는 공간에 맞추기 위해 크기를 키우거나 줄이는 방법을 설정하는 속성이다.

>[모질라 flexbox 개념 설명](https://developer.mozilla.org/ko/docs/Web/CSS/CSS_Flexible_Box_Layout/Basic_Concepts_of_Flexbox){:target="_blank"}

- 아이템 간 공간 배분과 정렬 기능을 제공하기 위한 1차원 레이아웃 모델이다.
  - FlexBox를 1차원이라고 칭하는 것은, 레이아웃을 다룰 때 한번에 하나의 차원(행이나 열)만을 다룬다는 뜻이다.
- 문서의 영역 중에서 FlexBox가 놓여있는 영역을 flex 컨테이너라고 부른다.<BR>flex 컨테이너를 생성하려면 영역 내의 컨테이너 요소의 display 값을 flex로 지정해야한다.<BR>이 값이 지정된 컨테이너의 일차 자식(direact children) 요소가 flex 항목(item)이 된다.

<BR><BR>

# 2. FlexBox의 두 개의 축

<BR>

## 2-1. 주축 (main axis)

주축은 `flex-direction`에 의해 정의되며 4개의 값을 가진다.
- `row` : 왼쪽에서 오른쪽
- `row-reverse` : 오른쪽에서 왼쪽
- `column` : 위에서 아래
- `column-reverse` : 아래에서 위

<BR>

## 2-2. 교차축 (cross axis)

교차축은 주축에 수직한다.

- row, row-reverse 라면 교차축은 열 방향을 따른다.
- column, column-reverse 라면 교차축은 행 방향을 따른다.

<BR><BR>

# 3. 시작선과 끝선

오른쪽, 왼쪽, 위, 아래에 종속적이지 않은 시작선(start)과 끝선(end) 개념을 쓴다.<BR>한글이라면 시작점을 왼쪽, 아랍어라면 오른쪽으로 잡을 수 있다.

<BR><BR>

# 4. flex 컨테이너 (default 값)

문서의 영역 중에서 flexbox가 놓여있는 영역을 flex 컨테이너라고 부른다. flex 컨테이너를 생성하려면 영역 내의 컨테이너 요소의 `display` 값을 `flex` 혹은 `inline-flex`로 지정한다.<BR>flex의 기본값(default)은 다음과 같다.
- 항목(item)은 행(row)으로 나열된다.
- 항목은 주축의 시작선에서 시작한다.
- 항목은 주 차원 위에서 늘어나지는 않지만 줄어들 수 있다.
- 항목은 교차축의 크기를 채우기 위해 늘어난다.
- `flex-basis`(item의 크기) value 는 `auto`로 설정된다.
- `flex-wrap` value는 `nowrap`으로 설정된다.

3번째 항목과 6번째 항목이 모순적으로 보이지만, 컨텐트 크기까지 줄어들다가 그 이상이 되면 컨테이너를 뚫고 나가게 되는 방식이다.

<BR><BR>

# 5. flex-wrap

flex 항목(item)들의 전체 크기가 컨테이너 크기보다 커졌을 때 행이나 열을 추가할지에 대한 속성
>value
  - `nowrap`
    - 추가하지 않고 넘치는대로 그대로 둔다.
  - `wrap` 
    - 컨테이너를 넘치는 항목을 새 행이나 열로 옮긴다.

<BR><br>

# 6. flex-flow

flex-direction과 flex-wrap을 같이 쓴다. (shorthand)
```css
flex-flow:[direction] [wrap]
```

<BR><BR>

# 7. flex-basis

[모질라 flex-basis](https://developer.mozilla.org/ko/docs/Web/CSS/flex-basis){:target="_blank"}

`flex-basis`는 플렉스 항목(item)의 초기 크기를 지정한다.

<div class="notice--primary" markdown="1">
주의!
- (auto 값을 가지지 않은) `flex-basis`와 `width`(flex-direction: column인 경우 height)값을 동시에 적용한 경우 `flex-basis`값을 우선한다.
</div>

>value

```css
/* <'width'> 지정 */
flex-basis: 10em;
flex-basis: 3px;
flex-basis: auto;

/* 원본 크기 키워드 */
flex-basis: fill;
flex-basis: max-content;
flex-basis: min-content;
flex-basis: fit-content;

/* 플렉스 아이템 내용 크기에 따라 조절 */
flex-basis: content;

/* 전역 값 */
flex-basis: inherit;
flex-basis: initial;
flex-basis: unset;
```
- `<width>` 에서 `auto`는 자신의 너비와 높이 값 속성을 사용한다.
- `content`는 말 그대로 컨텐트 크기 만큼 조절된다.
- 원본 크기 키워드 (주말에 추가 공부 예정)

# 11. justify-content

주축에 따라 flex item을 정렬하는 방식이다.

> value

- `flex-start`: 주축의 시작 지점부터 정렬
- `flex-end` : 주축의 끝 지점부터 정렬
- `center` : 주축의 중앙 정렬
- `space-between` : 여유 공간을 item들 사이로 균등 배분
- `space-around` : 여유 공간을 item을 둘러싸서 균등 배분
- `space-evenly` : 여유 공간을 아이템 사이, 시작, 끝 균등 배분
