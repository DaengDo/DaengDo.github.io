# 1. 배열 선언

- 배열 선언 방법은 아래 두 가지 방식이 있다.
  ```js
  let arr = new Array();
  let arr = [];

  // 초기 요소를 넣어주는 것도 가능하다
  let arr = ["사과", "오랜지", "복숭아"];
  ```
- 각 배열 요소에는 0부터 시작하는 숫자(index)가 매겨져 있다.<BR>이 숫자들은 배열 내 순서를 나타낸다.
- 배열 내 특정 요소를 아래 문법으로 얻을 수 있다.
  ```js
  let fruits = ["사과", "오렌지", "복숭아"];

  alert( fruits[0] ); // 사과
  alert( fruits[1] ); // 오렌지
  alert( fruits[2] ); // 복숭아
  ```
- 위의 방법으로 요소를 수정할 수 있다.
  ```js
  fruits[0] = '포도'; 
  // 배열이 ['포도', '오렌지', '복숭아'] 로 바뀐다.
  ```
- 새로운 요소를 배열에 추가하는 것도 가능하다.
  ```js
  fruits[3] = '자두';
  // 배열이 ['포도', '오렌지', '복숭아', '자두'] 로 바뀐다.
  ```
- `length`를 사용하면 배열에 담긴 요소가 몇 개인지 알아낼 수 있다.
  ```js
  let fruits = ['포도', '오렌지', '복숭아', '자두'];

  console.log(fruits.length);   // 4 
  ```
- 배열 요소의 자료형에는 제약이 없다.
  ```js
  let arr = ['포도', {name: "DaengDo"}, true, 12, function() {console.log("배열의 요소입니다.");},]

  // 인덱스가 1인 요소(객체)의 name 프로퍼티를 출력한다.
  console.log(arr[1].name); // DaengDo

  // 인덱스가 4인 요소(함수)를 출력한다.
  arr[4]();      // 배열의 요소입니다.
  ```
- 배열의 마지막 요소는 객체와 마찬가지로 쉼표로 끝날 수 있다.<BR>(trailing 쉼표)
  ```js
  let fruits = [
    "사과",
    "오렌지",
    "자두",
  ];
  ```
