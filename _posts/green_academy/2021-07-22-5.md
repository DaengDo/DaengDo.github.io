# 1. CSS에서의 크기 단위

> property value 로 쓰이는 크기 단위?

- 픽셀 (pixel, px) : 사각형의 점으로, 디지털 화상을 구성하는 기본적인 단위이다.
  - tip: 웹 브라우저 문서의 디폴트 폰트 크기는 16px<BR>(화면 비율 1920x1080도 픽셀단위)
- 퍼센트 (%)
- em : 부모 태그의 크기에 대한 상대적 단위이다.
  - ex: 부모 태그의 크기가 10px 일때 자손 태그에서 0.5em 을 하면 5px이 적용된다.
- rem : 최상위(root) 부모 태그의 크기에 대한 상대적 단위이다.<BR>em이 부모태그의 크기를 일일히 알고있어야 하는것에 비해 rem은 더 직관적이다.

# 3. 수업 중 나온 property
  
- `font-size` : 글씨 크기
- `width`, `height` : 너비, 높이
- `color` : 색상
- `background-color` : 배경 색상
- `margin`, `border`, `padding`
  - 위, 아래, 왼쪽, 오른쪽의 크기를 value로 조절 할 수 있다.
    - 4개의 값을 줄 경우 ↑ → ↓ ← (시계방향) 순서이다.
    - 2개의 값을 줄 경우 위/아래, 왼쪽/오른쪽 순서이다.

    ```
    h1 {

      margin : 0 0 0 0;    /* 12시부터 시계방향 순서 */

      border : 0 0;         /* 위/아래, 왼쪽/오른쪽 순서 */

    }
    ```
- `list-style` : ul와 ol 태그의 list 에 붙는 점이나 숫자의 스타일을 바꿀 수 있다.
- `box-sizing`
  - box-sizing의 value content-box와 border-box를 활용해 보았다.
  > content-box : width와 height가 content의 크기를 조절한다. (모든 태그의 디폴트 값)
  > border-box : width와 height가 전체 태그의 크기를 조절한다.(border 부터 content 까지)
- `user-select`
- `cursor` : 마우스 커서를 올렸을때 적용된다.
- `text-decoration`

#### 태두리 속성(property)

- `border-width` : border의 너비
- `border-style` : border의 스타일(점선,선 등)
- `border-color` : border의 색
- `border-radius`: value를 주면 시계방향(↖↗↘↙)순서대로 모서리를 둥글게 만들어준다.<BR> % 값을 주면 픽셀 단위보다 부드럽다.<BR>tip: 50% 값을 주면 원이 된다.

<BR>

#### 수업 중 나온 property value

- `auto` : 자동 값 지정, 수업에서는 자동으로 왼쪽 오른쪽 margin 값을 주어서 가운데 정렬을 하였다.

<BR><BR><BR>

# 4. sibling selector (동위 선택자)

> 들여쓰기가 같은 요소에서만 선택이 가능하다.

- adjacent sibling selector (인접 동위 선택자)
  - `+` : 바로 다음 요소를 선택자로 한다.<BR>ex) h1 + h2 는 h1다음에 오는 h2를 선택자로 한다.

- general sibling selector (일반 동위 선택자)
  - `~` : 앞 태그 뒤의 모든 요소를 선택자로 한다.<BR>ex) h1 ~ h2는 h1 다음 모든 h2 태그를 선택자로 한다.

<BR><BR><BR>

# 5. pseudo-class (의사 클래스)

> 의사 클래스는 선택하고자 하는 HTML 요소의 특별한 '상태'를 명시할 때 사용한다.

<BR>

## 반응 선택자

- `:active` : 사용자가 마우스로 태그를 클릭했을때 적용된다.
- `:hover` : 사용자가 포인터를 태그에 올렸을때 적용된다.

<BR>

## 링크 선택자

a 태그에 적용되는 선택자이다.

- `:link` : href 속성을 가지고 있는 a 태그를 선택한다.
- `:visited` : 방문한 링크가 있는 a 태그를 선택한다.

<BR><BR>
