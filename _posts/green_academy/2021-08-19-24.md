여러 컴포넌트를 만들어보며 아래 개념들을 익혔다.

## jsx

- React 의 요소(element)
  - React 내부에서 HTML 태그 구조를 만들 수 있는 요소

<BR>

## component

- jsx 를 반환하는 함수
- 각 컴포넌트에 최상위 부모 태그는 하나만 존재할 수 있다.

<BR>

## rendering

- html 요소, react 요소 등의 코드가 눈으로 볼 수 있도록 그려지는것을 렌더링(rendering)이라고 한다.
- `props` 혹은 `state`의 값이 변경되었을때 이를 react가 감지하고 값이 변경된 컴포넌트 부분을 새롭게 렌더링한다.

<BR>

## props

- component의 매개변수
- 화면의 렌더링 결과에 영향을 주는 값

<BR>

## state

함수형 컴포넌트가 useState라는 함수로 사용한 state 만을 배웠다. 클래스형 컴포넌트에 대해서는 배우지 않았다.

- component 내부에서 관리하는 변수
- 화면의 렌더링 결과에 영향을 주는 값
- 불변성(immutability)을 갖는다. (수정 불가능한 변수)
  - state 변경 함수로만 변경할 수 있다. <BR>(다른 방법으로는 변경 불가능)
