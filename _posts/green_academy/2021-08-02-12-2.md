# 4. 형 변환

## 4-1. 문자형으로 변환

- String(값) → 값이 문자열로 변환된다.
```js
let value = true;
console.log(String(value));
```
- "" + 값 → 값이 문자열로 변환된다.
```js
let value = true;
value = "" + vlaue;
console.log(typeof value);
```
- 값.toString() → 값이 문자열로 변환된다.
```js
let value = true;
value = value.toString();
console.log(typeof value);
```

<BR>

## 4-2. 숫자형으로 변환

- 수학 연산을 시도하면 문자를 숫자로 변환한다.
  ```js
  console.log("6" / "2");
  console.log("6" / "0");   // infinity
  console.log("abc" / "2"); // NaN
  ```
- Number() → 값이 숫자형으로 변환된다.
  ```js
  let num;

  console.log(Number("2"));       // 2
  console.log(Number(undefined)); // NaN
  console.log(Number(num));       // NaN
  console.log(Number(null));      // 0
  console.log(Number(true));      // 1
  console.log(Number(false));     // 0

  /* 앞뒤 공백을 제거하여 숫자로 변환 */
  console.log(Number(" 123 "));   // 123
  console.log(Number(" a23 "));   // NaN
  ```

<BR>

## 4-3. 불린형으로 변환

- Boolean() → 값이 불린형으로 변환된다
  ```js
  /* true 값으로 변환 */
  
  console.log(Boolean("abc")); // true
  console.log(Boolean(1));     // 0보다 큰 값은 true
  console.log(Boolean(Infinity)); // true

  /* false 값으로 변환 */

  console.log(Boolean("")); // false
  console.log(Boolean(0));  // false
  console.log(Boolean(undefined)); // false
  console.log(Boolean(null));      // false
  console.log(Boolean(NaN));       // false
  ```

# 5. 기본 연산자와 수학

## 5-1. 수학 연산자

- `+` : 덧셈 연산자 
- `-` : 뺄셈 연산자
- `*` : 곱셈 연산자
- `/` : 나눗셈 연산자
  ```js
  let a = 10, b = 5;
  console.log(a + b); // 15
  console.log(a - b); // 5
  console.log(a * b); // 50
  console.log(a / b); // 2
  ```

<BR>

## 5-2. 나머지 연산자 %

`%` 기호로 나타내며, `a % b` 는 a를 b로 나눈 후 그 나머지를 정수로 반환해준다.
```js
let a = 14, b = 5;
console.log(parseInt(a / b)); // 몫: 2
console.log(a % b);            // 나머지 : 4
```

<BR>

## 5-3. 거듭제곱 연산자 **

`**` 기호로 나타내며, `a**b`는 a를 b번 곱한 값이 반환된다.
```js
let a = 2, b = 3;
console.log(a ** b);  // 8
let c = 11, d = 2;
console.log(c ** d);  // 121
```

<BR>

## 5-4. 이항 연산자

- `+` : 문자열 결합
  ```js
  let s = "my" + "string";
  console.log(s); // mystring
  ```
- `+` : 문자열과 숫자의 결합
  ```js
  let sn = "1", n = 2;
  console.log(sn + n);  // "12", 문자열
  console.log(n + sn);  // "21", 문자열
  
  console.log(n + n + sn);  // "41", 문자열
  // 2 + 2 + "1" 형태이다.
  ```
- `+` : 문자열과 불린의 결합
  ```js
  console.log("false" + true);  // "falsetrue", 문자열
  console.log("" + true);       // "true", 문자열
  ```
- `+`를 제외한 수학 계산은 문자열을 숫자로 변환하여 계산한다.
  ```js
  let sn = "1", n = 2;
  console.log(n - sn);  // 1, 숫자
  console.log(n / sn);  // 2, 숫자
  ```

<BR>

## 5-5. 단항 연산자 +와 숫자형으로의 변환

`+` 는 단항 연산자로도 사용할 수 있다.

- 숫자에는 아무런 영향을 미치지 못한다.
- 숫자형이 아닌 피연산자는 숫자형으로 변환시킨다.
  ```js
  /* 숫자형에는 영향을 끼치지 못함 */
  let val = 1;
  console.log( +val ); // 1

  /* 숫자형이 아닌 피연산자를 숫자형으로 변환시킴 */
  let str = "2"
  console.log( +str ); // 2, 숫자

  console.log( +true ); // 1, 숫자
  console.log( +"" ); // 0, 숫자
  ```

<BR>

## 5-6. 할당 연산자 
`=` 기호로 나타내며, 할당(assignment) 연산자라고 부른다.
> 복합 할당 연산자
- `+=`, `*=` 과 같은 연산자를 사용하면 짧은 문법으로 연산을 수행할 수 있다.
  ```js
  let n = 2;

  n = n + 5;
  n += 5; // n = n + 5 와 같다.

  n = n * 2;
  n *= 2; // n = n * 2 와 같다.
  ```

<BR>

## 5-7. 증가/감소 연산자

숫자를 하나 늘리거나 줄일때 쓰이는 연산자이다.

- `++` : 증가(increment) 연산자
  ```js
  let counter = 2;
  counter++;  // counter = counter + 1과 같다.
  console.log( counter ); // 3
  ```

<BR>

- `--` : 감소(decrement) 연산자
  ```js
  let counter = 2;
  counter--;  // counter = counter - 1과 같다.
  console.log( counter ); // 1
  ```
