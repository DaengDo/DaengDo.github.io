리눅스는 아래 링크<BR>
[https://github.com/nvm-sh/nvm](https://github.com/nvm-sh/nvm){:target="_blank"}

- `nvm ls` or `nvm list`
  - (설치된) 노드 버전 목록
- `nvm install --lts`
  - 특정 노드 버전 설치
  - `nvm install v16.xx.x`
- `nvm use xx.xx.x`
  - xx.xx.x 의 노드 버전으로 전환
- `nvm uninstall xx.xx.x`
  - xx.xx.x 의 노드 버전 삭제
  - 현재 사용중인 버전은 삭제 불가능하므로 다른 버전으로 전환하고 삭제해야 함
