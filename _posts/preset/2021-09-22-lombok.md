자바 라이브러리


- [https://projectlombok.org/](https://projectlombok.org/){:target="_blank"}
  - lombok.jar 다운로드 후 커멘드 창에 `java -jar lombok.jar` 입력시 스프링이 설치된 경로 폴더에 lombok을 설치함(spring을 실행중이면 오류날 수 있음)
  - lombok.jar 파일을 sts 폴더에 복사하고 SpringToolSuite4.ini 파일에 `-javaagent:D:\sts-4.11.1.RELEASE\lombok.jar` 이 자동으로 추가 
- `spring initializer`로 웹에서도 할 수 있다.

new spring starter project 로 lombok test 프로젝트 시작
`lombokproject.png` 참고

<BR>

테스트


프로젝트 생성 후 소스파일 패키지에 Member 클래스 작성

```java
package com.example.testlombok;

import lombok.Data;

@Data // lombok annotation 적용되는지 채크
public class Member {
	private int id;
	private String name;
}
```
