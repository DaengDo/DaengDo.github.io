# keyboardshortcuts(JSON)

- `F1` 누른 후 <br>`Preferences:Open keyboard Shrotcuts(JSON)` 파일 진입

```
// Place your key bindings in this file to override the defaults
[
  {
    "key": "ctrl+down",
    "command": "-scrollLineDown",
    "when": "textInputFocus"
  },
  {
    "key": "ctrl+up",
    "command": "-scrollLineUp",
    "when": "textInputFocus"
  },
  {
    "key": "ctrl+down",
    "command": "terminal.focus",
    "when": "!terminal.focus"
  },
  {
    "key": "ctrl+up",
    "command": "workbench.action.focusActiveEditorGroup",
    "when": "terminalFocus"
  },
  {
    "key": "ctrl+down",
    "command": "-workbench.action.terminal.navigationModeFocusNext",
    "when": "accessibilityModeEnabled && terminalA11yTreeFocus && terminalProcessSupported || accessibilityModeEnabled && terminalFocus && terminalProcessSupported"
  },
  {
    "key": "ctrl+up",
    "command": "-workbench.action.terminal.navigationModeFocusPrevious",
    "when": "accessibilityModeEnabled && terminalA11yTreeFocus && terminalProcessSupported || accessibilityModeEnabled && terminalFocus && terminalProcessSupported"
  }
]
```
- ↑ 에디터 창에서 스크롤을 컨트롤 위아래 방향키로 움직이는걸 포기하고<BR>대신에 컨트롤 위 아래 방향키로 편집과 터미널 포커스를 옮기는 것으로 변경함

```
  {
    "key": "ctrl+shift+oem_7",
    "command": "explorer.newFile",
    "when": "!editorFocus"
  },
  {
    "key": "ctrl+oem_7",
    "command": "explorer.newFolder",
    "when": "!editorFocus"
  }
```
- ↑ 에디터 포커스가 아닐때 ctrl + ' (; 오른쪽에 있는 키) 누르면 익스플로러창 폴더 생성
- ↑ 에디터 포커스가 아닐때 ctrl + shift + ' (; 오른쪽에 있는 키) 누르면 익스플로러창 파일 생성
